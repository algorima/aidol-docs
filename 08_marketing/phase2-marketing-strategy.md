## 1. 마케팅 방향성

"나도 만들어보고 싶다" 욕구 자극 → 유저 참여 유도

## 2. 활용 가능한 기능 (Phase 1 deployed 기준)

| Sprint | 기능 | 마케팅 콘텐츠 활용 |
|--------|------|-------------------|
| Sprint 1 | 캐스팅, 멤버 생성, 포지션 배정, 그룹 완성 | 스피드런 영상, 그룹 결과물 쇼케이스 |
| Sprint 2 | 관계 설정, 관계 지도, 캐릭터 서사 | 관계성 콘텐츠, 멤버 간 케미 |
| Sprint 3 | AI 채팅, 수신함, 1:1 대화 | 채팅 스크린샷, 대화 스니펫 |
| 별도 제작 | 춤/노래 영상 (미진 담당) | 퍼포먼스 숏폼, 챌린지 |

## 3. 채널별 실행 전략

### 3.1 틱톡 / 인스타 릴스 - 신규 유저 유입

**목표 KPI**: 콘텐츠당 조회수 1만+ / 유저 자발 콘텐츠 주 10개+

**핵심 원칙 (2026 틱톡 알고리즘 기반)**
- 첫 2초 안에 훅: 63%의 바이럴 영상이 3초 내 시청자를 잡음
- 영상 길이: 21~34초 최적, 7~15초 마이크로 모먼트도 효과적
- 트렌드 사운드 사용 시 노출 3배 증가
- 업로드 빈도: 주 3~5회 (일관성 중요)
- 완주율이 가장 중요한 알고리즘 시그널

**훅 공식**

| 훅 유형 | 예시 | 효과 |
|---------|------|------|
| Before/After Snap | (텍스트 입력) → 완성된 그룹 퍼포먼스 | 변화 임팩트 |
| Bold Claim | "30초 만에 5인조 아이돌 그룹 만들었는데" | 호기심 유발 |
| Pattern Interrupt | 완성된 아이돌이 설명 없이 갑자기 등장 | 스크롤 멈춤 |
| POV/Relatable | "POV: 최애가 졸업해서 AI로 새 그룹 만든 사람" | 팬덤 공감 |

**콘텐츠 포맷**

| 콘텐츠 | 설명 | 제작 방식 |
|--------|------|----------|
| 스피드런 | 캐스팅 → 그룹 완성 15~20초 | 화면 녹화 + 트렌드 사운드 |
| 퍼포먼스 클립 | 완성된 그룹의 춤/노래 장면 | 미진 제작 영상 편집 |
| 결과물 플렉스 | 완성 그룹 베스트 장면 7~10초 루프 | 화면 캡처 + 음악 |
| 비교 | "이 그룹 실제 vs AI?" | AI 프로필 이미지 활용 |
| 관계성 | "두 멤버에게 서로를 물어봤더니..." | Sprint 2 관계 지도 + 채팅 |
| 채팅 반응 | 멤버와 대화 중 재밌는 답변 | Sprint 3 채팅 스크린샷 |
| 이상형 그룹 | 매번 다른 컨셉 (Y2K, 걸크러시, 청량 등) | 멤버 생성 과정 녹화 |

**하지 말 것**
- 느린 인트로, "안녕하세요 오늘은~"
- 로고 먼저 보여주기
- 텍스트 카드로 시작
- 튜토리얼처럼 보이는 시작

**운영**
- 브랜드 워터마크 노출
- 결과 중심 콘텐츠
- 틱톡 Creative Center에서 트렌드 사운드 확인

### 3.2 트위터 / X - 팬덤 바이럴

**목표 KPI**: 팔로워 1,000명

아이돌 팬덤 문화와 관련 소비가 가장 활발한 채널.

| 콘텐츠 | 예시 | 바이럴 포인트 |
|--------|------|-------------|
| 제작 그룹 | 잘 만든 그룹으로 서비스 소개 (프로필 + 퍼포먼스 + 서사) | 동기 부여, 모방 욕구, 팬심 자극 |
| 참여 유도 | "내 이상형은?" | 댓글, RT 유도 → 앱 진입 |
| 관계성 | "정서적으로 의지하는 관계의 두 멤버에게 서로를 물어봤더니..." | "나도 이런 관계성 만들어볼까?" 상상 |
| 채팅 | 멤버와 나눈 대화 중 베스트 장면 공유 | 공감, 재미 |

**운영**
- 하루 1~2개 포스팅 루틴 유지
- RT, 인용 적극 활용
- 자체 해시태그 운영

### 3.3 커뮤니티 - 타겟 유저 직접 공략

**목표 KPI**: 유저 자발 콘텐츠 주 10개+

체험 후기를 자연스럽게 올리는 방식.

| 커뮤니티 | 접근 방식 | 주의 |
|---------|----------|------|
| 더쿠 | 자연스러운 체험 후기 | 광고 티가 나지 않게 |
| 디시 AI 채팅 갤러리 | 자연스러운 비교 후기 | 비방 아닌 차이점 공유 |
| Reddit | 영문 콘텐츠로 해외 유저 접점 | 영어 포스팅 별도 준비 |

**운영**
- 채널 당 1개의 포스팅
