# 문제 정의 (Problem)

## 한 줄 요약

> **"정말 좋아하는 무언가가 있는데, 그 이야기를 진심으로 들어줄 사람은 어디에도 없었던 순간"**

---

## Problem 1: 팬의 외로움 (The Unspoken Loneliness of a Fan)

### 창업자 스토리

> "저는 20년 넘게 팬으로 새벽부터 콘서트 티켓을 위해 줄 서고, 공연장에서 면봉만 한 아이돌을 바라보는 경험에 20만원씩 지불했습니다. 
> 그립톡, 응원봉 키링 등 굿즈를 유통하고 판매하며 깨달은 것은, 팬덤이 단순한 상품 소비가 아니라 
> **'내가 사랑하는 존재를 일상에서도 느끼고 싶다'**는 본질적인 욕구라는 것입니다."
> — 이소연 대표

### 팬들의 고충

| 문제 | 현실 |
|------|------|
| **접근성** | 영통팬싸 당첨률 ~1%, 가격 200-400만원 |
| **물리적 한계** | BTS 1억 팬 × 3분 통화 = 평생 90%가 대화 불가 |
| **팬미팅 비용** | $200+ (해외 팬은 항공비 별도) |
| **커뮤니티 피로** | 낯선 팬들과의 배타적 문화, 때로는 배척 |
| **언어 장벽** | 번역기에만 의존, 글로벌 팬 소외 |

### 미국 Z세대 K-pop 팬의 현실

- 열정을 이해하지 못하는 주변 친구들
- 한국 팬덤 문화의 배타성 → 긴장감, 소외감
- **결론**: 좋아하는 걸 마음껏 이야기할 수 있는 편안한 공간과 친구를 원함

## Problem 2: 미개척 시장 (The Untapped Market)

### 거대한 숫자들

| 지표 | 수치 | 의미 |
|------|------|------|
| K-pop 시장 규모 | **$12B** (약 16조원) | 연간 |
| 글로벌 K-pop 팬 | **1.5억 명+** | |
| 슈퍼팬 연간 지출 | **$1,422** | 평균 |
| 플랫폼 외부 팬 비율 | **93%** | Weverse MAU 기준 |
| 방치된 소비 잠재력 | **$21.9B/년** | |

### 개인화의 부재

- **공식 채널의 한계**: 팬 수백만 명당 1개의 게시물
- **일방적 소통**: "소속사의 일방적 소통과 의견 미반영"에 대한 부정적 감정 **67%** (자체 분석)
- **결과**: 심리적 거리감 심화, 팬덤 활동 저해

## Problem 3: 소속사의 딜레마

### 구조적 한계

| 문제 | 영향 |
|------|------|
| **물리적 한계** | "아티스트는 하루 24시간, 몸은 하나 뿐" → IP 확장 제약 |
| **데이터 부재** | 팬 데이터 활용 불가 → 전략 실패 시 **수십억 원대 손실** |
| **아티스트 번아웃** | 고비용 오프라인 확장의 한계 |
| **기술 도입 장벽** | 기존 캐릭터 AI 플랫폼의 데이터 불투명성, 법적/윤리적 리스크 |

## Problem 4: 기존 솔루션의 한계

### For Fans

| 서비스 | 문제점 |
|--------|--------|
| **Bubble** | 일방향 메시지, 양방향 대화 불가, 개인화 없음 |
| **Character.AI** | K-pop 특화 없음, 팬덤 언어 미지원, 내가 만든 캐릭터 아님 |
| **PLAVE** | 사용자가 직접 생성 불가 |
| **온라인 커뮤니티** | 배타적 문화, 언어 장벽, 피로감 |

### For Agencies

| 문제 | 현재 상황 |
|------|-----------|
| 팬 데이터 수집 | 불가능 |
| 전략 실패 리스크 | 수십억 손실 |
| IP 확장 | 물리적 한계 봉착 |

## 검증 데이터

### 정성적 검증

1. **AI Boyfriend 실험 (300명)**
   - 75%가 AI에게 진심 어린 애정 표현
   - → 감정적 연결이 가능하다

2. **BTS AI 실험 (미국 시장)**
   - K-pop 팬들이 **15분 만에 깊은 이야기 공유**
   - → 가장 열정적인 시장 = K-pop 팬덤

3. **Virtual Idol 실험 (자체 IP)**
   - 500K 조회, 500 가입 (5일)
   - → 라이선스 문제 해결 + 바이럴 가능성

### 정량적 검증

- TikTok 조회수: **500,000+** (5일)
- 리드 수집: **5,000+**
- 가입자: **500+**
