# 문제 정의 (Problem)

## 핵심 문제

K-pop 팬들은 좋아하는 아이돌과 **진짜 소통**하고 싶지만, 현실적으로 불가능하다.

## 구체적 Pain Points

### 1. 접근성의 한계
- BTS 팬 1억 명+ → 각 팬과 3분씩 통화해도 평생 90%는 대화 불가
- 팬사인회 당첨 확률: 0.01%
- 팬미팅 비용: $200+ (해외 팬은 항공비 별도)

### 2. 기존 솔루션의 한계

| 서비스 | 문제점 |
|--------|--------|
| Bubble | 일방향 메시지, 양방향 대화 불가 |
| Character.AI | K-pop 특화 없음, 내가 만든 캐릭터 아님 |
| PLAVE | 가상 아이돌이지만 사용자가 직접 생성 불가 |

### 3. 감정적 니즈
- 팬들은 아이돌에게 **정서적 지지**를 받고 싶어함
- 실제 검증: 미국 K-pop 팬들이 BTS AI에게 15분 만에 속마음 털어놓음
- 75%의 사용자가 AI 캐릭터에게 애정 표현

## 요약

**팬덤은 강력한 감정적 연결인데, 기존 서비스는 이를 충족시키지 못한다.**
