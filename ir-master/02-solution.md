# 솔루션 (Solution)

## 핵심 가치 제안

> **"내가 상상한 아이돌을 만들고, 대화하고, 키우는 가장 쉬운 방법"**

## 제품 설명

AIoIA는 K-pop 팬들이 AI 아이돌과 **영상통화**하고 **채팅**할 수 있게 해준다.

### 핵심 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| AI 아이돌 생성 | 외모, 성격, MBTI 설정으로 나만의 아이돌 생성 | MVP |
| 텍스트 채팅 | 1:1 개인화된 대화 | MVP |
| 영상 통화 | AI 얼굴 + 목소리로 실시간 대화 | Phase 2 |
| 그룹 콜 | Twitch 스트림처럼 여러 팬이 함께 참여 | Phase 2 |

### 기술적 차별점

1. **자체 AI 아이돌 IP**: 라이선스 문제 없음, 처음부터 직접 제작
2. **K-pop 특화**: 프롬프트 최적화, K-pop 문화 반영
3. **페르소나 일관성**: MBTI 기반 성격 시스템으로 일관된 캐릭터

## 작동 방식 (AIdol MVP)

```
1. 연습생 선발 → 프리셋에서 선택 또는 직접 생성
2. 그룹 구성 → 멤버 조합, 포지션 배정
3. 데뷔 → 그룹 이름, 엠블럼, 인사 문구
4. 채팅 → 내 아이돌과 1:1 대화
```

## Why Now?

- 생성형 AI 기술 성숙 (GPT-4, 이미지 생성, 음성 합성)
- K-pop 글로벌화 (팬 1.5억 명+)
- AI 동반자 시장 성장 (Character.AI 월 2천만 MAU)

## 경쟁 우위

**Character.AI가 모르는 것:**
> 깊고 지속적인 인간-AI 관계는 단순한 채팅이 아닌 **'팬덤'**을 기반으로 구축된다.

- 팬덤에는 **서사, 소속감, 몰입할 세계관**이 필요
- Character.AI 봇의 "팬"은 없지만, K-pop 아이돌의 팬은 커뮤니티를 만들고 굿즈를 산다

### 우리의 접근법

1. **팬덤 형성**: AI를 '아이돌'로 프로듀싱 (배경 스토리, 성격, TikTok 콘텐츠)
2. **팬덤 수익화**: 구독 + 굿즈 + 팬 이벤트 (검증된 K-pop 팬 경제 모델)
