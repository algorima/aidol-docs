# AIdol IR 마스터 문서

이 폴더는 모든 IR/지원서의 **원본 소스**입니다.
각 지원서는 여기서 내용을 복사해서 양식에 맞게 수정합니다.

## 📁 구조

```
ir-master/
├── README.md              # 이 파일
├── 00-company-overview.md # 회사 기본 정보
├── 01-problem.md          # 문제 정의
├── 02-solution.md         # 솔루션
├── 03-traction.md         # 트랙션/성과
├── 04-market.md           # 시장 분석
├── 05-business-model.md   # 비즈니스 모델
├── 06-team.md             # 팀 소개
├── 07-financials.md       # 재무 계획
├── 08-ask.md              # 투자 요청
└── mappings/
    ├── yc-mapping.md      # YC 지원서 매핑
    └── 초창패-mapping.md   # 초기창업패키지 매핑
```

## 🔄 사용법

1. 마스터 문서 내용 업데이트
2. 지원서 작성 시 매핑 파일 참고
3. 해당 섹션 복사 → 양식에 맞게 수정
