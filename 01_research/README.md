# 01. Research (사용자 리서치)

사용자 조사, 시장 분석, 경쟁사 리서치 문서를 관리합니다.

## 폴더 구조

- `user-interviews/` - 사용자 인터뷰 및 설문조사 결과
- `market-analysis/` - 시장 분석 및 트렌드 리포트
- `competitor-research/` - 경쟁사 분석

## 파일명 규칙

```
YYYYMMDD__주제__상태__vX.Y__소유자.md
```

예시:
- `20260204__user-interview-kpop-fans__final__v1.0__sy.md`
- `20260201__kpop-market-analysis-2026__review__v0.2__yw.md`
