## 분석 전제 및 신뢰도 한계

이 녹취록은 피실험자(은재님 언니)의 직접 발화가 아닌, 은재님이 소연님에게 UT 결과를 구두 보고하는 2차 전달(second-hand report) 형식이다. 따라서 다음 한계를 명시한다.

1. 피실험자의 정확한 원문 발화를 알 수 없다. 은재님의 기억과 언어로 재구성된 내용이다.
2. 은재님이 인용부호(" ")로 감싼 발화는 원문에 가까운 재현으로 간주하나, 완전한 verbatim이 아닐 수 있다.
3. 은재님의 해석과 피실험자의 실제 의도가 일치하지 않을 수 있다.
4. 모든 발화 인용은 "(은재님 전달, 타임스탬프)"로 표기하여 2차 보고임을 명시한다.
5. 미팅 초반(00:00~08:00)은 소연님의 투자 미팅 근황 공유로, UT 분석과 무관한 내용이다. 분석에서 제외한다.

### 3.2 감정 전환점

전환점 1 (부정→강한긍정): AI 이미지 생성 결과 확인 순간. "기대를 안 하고 있다가" → 자세 교정 + "상상하던 그대로". 트리거: 결과물 퀄리티가 기대치를 초과.

전환점 2 (긍정→강한부정): 그룹명 설정 화면. 이미지 생성의 높은 몰입 직후 책임감 과중으로 이탈 충동. 트리거: 수정 불가 인식 + 영구성 부담.

전환점 3 (부정→긍정): 관계성 설정. "이걸 꼭 할 필요가 있을까" → 설명 후 전체 완주. 트리거: 은재님의 구두 설명(채팅 반영 예고).

### 3.3 Peak-End 분석

Peak moment: AI 이미지 생성 결과 확인 순간(UT 중반). 신체 반응(자세 교정)을 동반한 강한 긍정 감정. 이 서비스의 핵심 가치 발견 순간.

End moment: 하이라이트 콘텐츠 탐색 종료. 타 그룹 낯섦으로 인한 소극적 참여. 본인 그룹 콘텐츠를 기다리는 상태로 종료. 중립적 긍정(0/+).

[해석] Peak-End Rule 적용 시, 이미지 생성 AHA 순간의 강렬한 인상이 전체 경험 기억을 주도할 가능성이 높다. End가 중립적이므로 전체 인상은 "신기하고 재미있었던 경험"으로 기억될 가능성이 높다.

### 5.2 Phase 3-5: 주제 도출

#### 주제 1. "내 손으로 만든 것"에 대한 강한 집착

정의: 이 사용자는 프리셋이나 완성된 결과물보다 자신이 직접 설계하고 창작한 것에 훨씬 강한 관여도를 보인다. 프리셋 무시(V001, V002), MBTI 1% 단위 조작(V003), 구체적 프롬프트 작성(V005)이 모두 같은 동기에서 나온다.

핵심 인사이트: 창작 과정 자체가 이 사용자에게는 콘텐츠다. 결과물보다 과정이 만족의 원천.

대표 발화:
- "빨리 생성하는 경험을 더 하고 싶다" (V002)
- "이 멤버의 얼굴을 미리 상상을 하고 어떤 성격이 어울릴지 생각하면서 MBTI를 고민했어요" (V004)
- "본인이 상상하던 그대로 나왔다고" (V005)

#### 주제 2. AI 이미지 생성 결과물이 애착 형성의 핵심 트리거

정의: 이 서비스에서 사용자 애착이 형성되는 가장 강력한 순간은 AI 이미지 생성 결과가 기대를 초과하는 순간이다. 그 이전까지의 텍스트 입력(이름, 서사, MBTI)은 준비 단계이고, 이미지가 나오는 순간 캐릭터가 "실체화"된다.

핵심 인사이트: 이미지 생성 품질과 속도는 이 서비스의 핵심 가치 전달 지점이다. 재생성 시 이전 이미지 손실 문제는 이 몰입을 방해하는 가장 직접적인 UX 장애물이다.

대표 발화:
- "기대를 안 하고 있다가 이제 결과물이 딱 나오니까 엄청 집중도 확 올라가면서" (V005)
- "자세도 누가 갑자기 고쳐 앉으면서" (V006)
- "이미지를 재생성하면 기존 이미지가 없어져 버리니까 이것보다 좋은 결과가 나올지 아님 더 악화가 될지 고민이 돼" (V008)

#### 주제 3. 관계성/콘텐츠 기능의 가치는 설명 없이 자발적으로 인지되지 않음

정의: 관계성 설정과 하이라이트 콘텐츠는 사용자가 UI만 보고는 "왜 이걸 해야 하는지" 이해하지 못한다. 가치 설명이 선행된 후에야 참여가 발생한다.

핵심 인사이트: 관계성과 콘텐츠의 가치 제안(채팅 반영, 스토리 생성)이 UI 내에서 명시적으로 전달되어야 한다. 현재는 구두 설명에 의존하고 있어 제품 단독으로는 작동하지 않는다.

대표 발화:
- "처음에 관계성을 설정해야 된다고 얘기를 했을 때는 그 필요성을 처음에 이렇게 느끼지 못해 했어요" (V016)
- "이게 내가 즐길 수 있을까" (V016)
- "관계성이 나중에 채팅이 반영이 되고 이런 다른 요소들이 이렇게 될 거다 라고 설명을 해주니까 그때서야 이렇게 좀 필요성을 느끼면서" (V017)

#### 주제 4. "내 그룹"에 대한 소속감과 콘텐츠 기대

정의: 사용자는 타인이 만든 그룹의 콘텐츠에는 관심이 없지만, 직접 만든 그룹의 콘텐츠에는 강한 기대를 보인다. 이 차이는 창작 투자(시간, 상상력)에서 비롯된 소속감이다.

핵심 인사이트: 애착은 창작 과정의 깊이에 비례한다. 그러나 현재 하이라이트 콘텐츠는 "내 그룹"보다 "타 그룹"이 먼저 노출되는 구조여서 이 애착을 활성화하지 못한다.

대표 발화:
- "본인이 만든 그룹의 콘텐츠는 언제 볼 수 있느냐 이러면서 먼저 그걸 더 기다렸던 것 같아요" (V021)
- "너무 많은 그룹을 보기보다는 적은 그룹 안에서 콘텐츠를 많이 접하면서 더 친숙해지는 게 좋을 것 같다고" (V025)

#### 주제 5. 해외 시장 및 다양성에 대한 자발적 문제 제기

정의: 사용자는 AI 아이돌 서비스가 젠더 이분법을 벗어나야 글로벌 시장에서 경쟁력이 있다고 자발적으로 제안했다. 이는 단순한 기능 요청이 아니라 서비스의 정체성과 시장 전략에 대한 인사이트다.

대표 발화:
- "성별이 이분법적으로 이렇게 적혀 있는 게 조금 걱정이 된다" (V027)
- "AI라서 오히려 이런 게 또 좀 더 자유롭고 도전적인 걸 만들고 싶은 사람이 있을 수 있겠다" (V014)

- 포지션별 기본 관계성 UI의 의미 파악 실패 (M04).

흐름 멘탈모델 간극:
- 재생성 시 이전 이미지 복구 불가 (M01). 사용자는 "실행 취소" 또는 "비교 선택" 플로우를 기대.

가치 멘탈모델 간극:
- 타 그룹 콘텐츠가 왜 홈에 있는지 이해 안 됨 (M05). "나는 내 그룹 콘텐츠만 보고 싶다"는 멘탈모델과 충돌.

### 9.3 비유/메타포 분석

직접적인 타 서비스 비유 발화는 이 녹취록에서 관찰되지 않음. 단, 은재님이 언급한 "인스타 피드" 비유(29:19 은재님 발화, 피실험자 발화 재현)는 피실험자가 콘텐츠 소비 방식을 인스타그램 피드와 유사하게 기대함을 시사한다.

[추론] 피실험자가 AI 이미지 생성 도구(Midjourney 등)를 사용해본 사람으로서, 이미지 생성 결과 관리(히스토리, 비교)에 대한 기대 수준이 일반 사용자보다 높을 가능성이 있다.

### 9.4 간극 해소 방향

M01 (재생성 손실): 시스템 변경 필요. 이전 이미지 히스토리 저장 또는 재생성 전 "현재 이미지를 저장하겠습니까?" 확인 모달.

M02 (그룹명 영구성 인식): UX 문구 변경으로 해결 가능. "나중에 수정할 수 있습니다" 안내 문구 추가.

M03, M04 (관계성 가치 불명확): 온보딩 문구 추가. 관계성 화면 진입 시 "이 설정은 멤버 간 채팅과 콘텐츠에 반영됩니다" 명시. UX 문구 변경으로 해결 가능.

M05 (타 그룹 우선 노출): 기능 구조 변경 필요. 홈 화면에서 "내 그룹" 콘텐츠를 우선 노출하는 개인화 구조.


## 10. Norman's 7 Stages of Action

### 10.1 태스크: AI 이미지 생성 및 재생성

| Stage | 관찰 | 문제 여부 |
|-------|------|----------|
| 1. Goal | 머릿속의 멤버 외모를 실제 이미지로 만들고 싶다 | 없음 |
| 2. Plan | 프롬프트에 외모를 구체적으로 작성한다 | 없음 |
| 3. Specify | 이미지 생성 버튼을 찾아 클릭 | 없음 |
| 4. Perform | 상세 프롬프트 작성 후 생성 실행 | 없음 |
| 5. Perceive | 이미지 결과물 표시됨 | 없음 (만족) |
| 6. Interpret | "상상하던 그대로 나왔다" — 올바른 해석 | 없음 |
| 7. Compare | 얼굴은 만족, 스타일링 불만족 → 재생성 고려 | 문제 있음 |

Stage 7에서 재생성을 고려하나, 이전 이미지 손실 공포로 인해 재생성 실행을 주저. Gulf of Evaluation: "재생성하면 지금 이미지가 사라질까?" 확인 불가.

### 10.2 태스크: 관계성 설정

| Stage | 관찰 | 문제 여부 |
|-------|------|----------|
| 1. Goal | 멤버 간 관계를 설정하고 싶다 (기대 미형성) | 목표 자체가 불명확 |
| 2. Plan | 이걸 왜 해야 하는지 모르겠다 | 문제 있음 |
| 3. Specify | 관계성 화면 진입 시 첫 화면 의미 파악 실패 | 문제 있음 |
| 4. Perform | 은재님 설명 후 관계 설정 실행 | 외부 도움 필요 |
| 5. Perceive | 관계성 설정 완료 | 없음 |
| 6. Interpret | 설정이 나중에 채팅에 반영된다는 것을 사후에 이해 | 문제 있음 |
| 7. Compare | 전원 완주. 복합 관계성 니즈 발생 | 부분 만족 |

### 10.3 Gulf of Execution (실행의 간극)

- 관계성 화면에서 "이걸 왜 해야 하는지" 몰라 목표 형성 자체가 실패. "이게 내가 즐길 수 있을까" (V016)
- 포지션별 기본 관계성 화면의 UI를 한 번에 이해하지 못함 (V020). 무엇을 해야 할지 파악 실패.

### 10.4 Gulf of Evaluation (평가의 간극)

- 재생성 버튼 클릭 시 현재 이미지가 사라지는지 여부를 사전에 확인할 수 없음. "이걸 누를지 말지도 되게 고민을 이렇게 많이 망설여지는 것 같았어요" (V008)
- 그룹명 설정 후 수정 가능 여부를 알 수 없음. 이 불확실성이 이탈 충동으로 이어짐 (V010).

### 11.4 3 Levels 종합

Visceral 우세: AI 이미지 생성 결과물의 품질이 가장 강렬한 반응을 유발했다. 이 서비스에서 Visceral 레벨은 명확히 작동하고 있다.

Behavioral: 신규 생성 플로우는 원활하나, 재생성 손실과 그룹명 영구성 인식이 행동을 방해하는 주요 마찰이다.

Reflective: "내 그룹 콘텐츠를 기다린다", "AI라서 더 자유로울 수 있다"는 발화에서 장기적 관계와 서비스 가능성에 대한 성찰이 시작됨. 그러나 아직 얕은 수준. Reflective 레벨까지 도달하게 하는 추가 경험(내 그룹 콘텐츠 제공)이 필요.


## 12. Hypothesis Validation

### 12.1 가설: "캐스팅 + 관계성 설정만으로 사용자가 AI 아이돌 그룹에 애착을 형성하고, 콘텐츠 소비 및 결제 의향으로 이어지는가"

가설을 3개의 하위 가설로 분해하여 검증한다.

#### H1: 캐스팅 + 관계성 설정으로 애착이 형성된다

판정: 부분 검증 (조건부)

지지 근거:
- [V005, V006] 이미지 생성 결과물에서 강한 몰입 발생 ("상상하던 그대로 나왔다", 자세 교정). 이미지 생성을 통한 창작 과정에서 애착의 시드가 형성됨.
- [V003, V004] MBTI 슬라이더 1% 단위 신중 조작. 깊은 창작 투자.
- [V015] 관계성 설정 전원 완주. 저항 후에도 완료.
- [V021] "본인이 만든 그룹의 콘텐츠는 언제 볼 수 있느냐". 내 그룹에 대한 소속감 형성 확인.

반박 근거:
- [V016] 관계성 기능 자체의 가치를 자발적으로 인지하지 못함. 은재님의 설명 없이는 참여하지 않았을 가능성 높음.
- [V012] 완성 후 추가 CTA(이메일 입력, 타 그룹 생성) 미반응. 애착이 행동으로 전환되지 않음.

판정 근거: 애착은 형성되었으나, 그 원천은 "캐스팅 + 관계성 설정" 전체가 아니라 주로 "AI 이미지 생성 결과물의 기대 초과" 순간에 집중된다. 관계성 설정은 설명 없이는 애착 형성에 기여하지 못한다.

조건: AI 이미지 생성 품질이 기대를 초과할 때. 신규 생성(프리셋이 아닌)을 선택한 사용자에게. 관계성 가치 설명이 선행된 경우.

#### H2: 콘텐츠 소비로 이어진다

판정: 미검증 (조건 불충족)

지지 근거:
- [V021] 내 그룹 콘텐츠를 기다림. 소비 의향 존재.

반박 근거:
- [V021, V022] 타 그룹 콘텐츠 소비 없음. "너무 초면이고 누가 있는지 모르는 상태에서 콘텐츠를 소비하기가 쉽지 않다".
- [V024] 콘텐츠 길이가 너무 짧아 아쉬움. 현재 콘텐츠가 소비 욕구를 충족시키지 못함.

판정 근거: "내 그룹" 콘텐츠가 아직 제공되지 않은 상태에서 콘텐츠 소비 행동을 관찰할 수 없었다. 타 그룹 콘텐츠는 소비되지 않았다. 가설 검증을 위해서는 사용자가 직접 만든 그룹의 콘텐츠를 받아본 후 반응을 관찰해야 한다.

#### H3: 결제 의향으로 이어진다

판정: 미검증 (데이터 없음)

이 녹취록에서 결제, WTP, 가격에 대한 언급이 전혀 없다. 가설 검증 불가. 후속 UT에서 명시적으로 질문 필요.

### 12.2 가설 외 발견 (Emergent Findings)

EF01: 신규 생성과 프리셋 캐스팅은 서로 다른 사용자 니즈를 충족한다. 이 사용자는 프리셋을 전혀 사용하지 않았다. 세그먼트(창작 성향 vs 팬 성향)에 따라 선호 경로가 다를 가능성이 높다.

EF02: 그룹명 설정이 유일한 이탈 위험 지점이다. 현재 가설 프레임에 포함되지 않은 이탈 요인.

EF03: 젠더 이분법에 대한 자발적 문제 제기. 글로벌 시장을 고려한 제품 정체성 이슈로, 단일 사용자 발언이나 K-POP 트렌드(XLOVER 등 젠더레스 그룹)와 맥락이 일치한다.

EF04: 타 그룹 콘텐츠 소비에 대한 진입 장벽. "내 그룹" 우선 경험 설계가 필요하다는 아키텍처 수준의 발견.

## 14. Cross-Analysis Summary

### 14.1 이 사용자의 핵심 인사이트 (Top 5)

1. AI 이미지 생성 결과물의 기대 초과가 애착 형성의 핵심 트리거다
   - 근거 방법론: Emotional Journey (전환점 1), Norman's 3 Levels (Visceral), Thematic Analysis (주제 2)
   - 대표 발화: "기대를 안 하고 있다가 이제 결과물이 딱 나오니까 엄청 집중도 확 올라가면서 이렇게까지 잘 나올 줄 몰랐다고 본인이 상상하던 그대로 나왔다고" (V005, 17:28)
   - 제품 시사점: 이미지 생성 품질과 속도는 P0. 재생성 히스토리 보존은 이 AHA 경험을 보호하는 핵심 UX.

2. 관계성 기능은 현재 단독으로 작동하지 않는다. 가치 설명이 UI에 내장되어야 한다.
   - 근거 방법론: Hypothesis Validation (H1 조건부), Mental Model Gap (M03), Norman's 7 Stages (Gulf of Execution)
   - 대표 발화: "처음에 관계성을 설정해야 된다고 얘기를 했을 때는 그 필요성을 처음에 이렇게 느끼지 못해 했어요" (V016, 26:49)
   - 제품 시사점: 관계성 화면 진입 시 "이 설정은 채팅과 콘텐츠에 반영됩니다" 문구 추가. UX 문구 변경으로 해결 가능.

3. 그룹명 설정은 현재 유일한 이탈 위험 지점이다.
   - 근거 방법론: Behavioral Sequence (이탈 지점), Norman's 7 Stages (Gulf of Evaluation), Pain/Gain
   - 대표 발화: "나중에 수정도 수정도 안 될 것 같아서 이때 약간 포기할까 조금 생각을 했었던 것 같아요" (V010, 19:27)
   - 제품 시사점: "나중에 수정 가능합니다" 문구 추가. 즉시 적용 가능. P0.

4. 사용자는 "내 그룹" 콘텐츠만 소비하고 싶다. 타 그룹 콘텐츠는 진입 장벽이 높다.
   - 근거 방법론: Thematic Analysis (주제 4), JTBD (Job D), Mental Model Gap (M05)
   - 대표 발화: "본인이 만든 그룹의 콘텐츠는 언제 볼 수 있느냐 이러면서 먼저 그걸 더 기다렸던 것 같아요" (V021, 27:45)
   - 제품 시사점: 콘텐츠 홈에 "내 그룹" 우선 노출. 타 그룹 탐색은 옵트인 방식으로.

5. 창작 과정 자체가 이 사용자에게는 핵심 콘텐츠다. 프리셋은 창작 욕구를 충족시키지 못한다.
   - 근거 방법론: JTBD (Job B), Behavioral Sequence (패턴), Proto-Persona
   - 대표 발화: "빨리 생성하는 경험을 더 하고 싶다" (V002, 09:17), "이 멤버의 얼굴을 미리 상상을 하고 어떤 성격이 어울릴지 생각하면서 MBTI를 이렇게 고민했어요" (V004, 10:17)
   - 제품 시사점: 신규 생성 경험의 깊이를 강화. 프리셋 캐스팅과 신규 생성의 진입 경로를 사용자 성향별로 최적화.

### 14.3 세그먼트 관점 해석

이 사용자는 A(팬 성향)와 B(창작 성향)의 혼합이며, B 성향이 더 두드러진다. AI 이미지 생성 도구 숙련자라는 점이 행동 전반에 영향을 미쳤다.

- 프리셋 무시와 신규 생성 직행: 창작 성향(B)의 전형적 행동.
- MBTI 1% 단위 신중 조작, 상세 서사 작성: 창작 성향(B)의 과몰입.
- 타 그룹 콘텐츠 무관심, 내 그룹 콘텐츠 기대: 팬 성향(A)이 창작한 대상에 적용됨.
- 이미지 프롬프트 숙련 작성: AI 숙련자 특성. 일반 사용자(AI 비숙련)와의 결과가 다를 수 있음.
- 젠더레스 시장 언급: K-POP 글로벌 시장 인식. 세그먼트 내에서도 특이한 시야.

[해석] AI 이미지 생성 숙련 사용자는 이미지 결과물에 대한 기대치가 높고, 재생성 히스토리 등 고급 기능 니즈도 높다. 일반 사용자 대상 UT와 병행하여 기준 비교가 필요하다.
