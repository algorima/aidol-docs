# SPRINT 1-2 - UT Sprint 요약: U10 최지윤

> Sprint 1 재참여자. U7(Sprint 1)로 캐스팅 경험 후, Sprint 1-2 구현 기능을 동일 세션 내 재경험.
> 예나 직접 생성 경험이 캐스팅 소유감 및 세계관 수정 전반에 개입 변수로 작용함.
> 수지의 사용자 인터뷰 자동 정리 프롬프트 가이드에 따라 정리.

## 피실험자 정보

- 세그먼트: **A그룹 (아이돌 관심/팬덤) — U7 Sprint 1 재참여자**
- 예나 직접 생성 경험 있음 (이전 Sprint)

---

## 0. 한 줄 핵심

캐스팅 소유감과 세계관 수정 반응 모두 예나 직접 생성 경험에 기반함. 세계관 수정은 제작으로 인식되나 실제 행동은 피로/회피. 채팅만 독립적 기대 신호로 확인.

---

## 1. 가장 인상 깊은 사용자 발화 (Raw Voice)

> "후보 수가 (지난번보다) 적어서… 오히려 덜 힘든 것 같아요. 선택권이 얼마 없으니까요."

> "이 멤버 구성이… 뺏기고 싶지 않아서요."

> "예나를 직접 만들어서 그런 것 같아요. 예전엔 상관없었는데, 예나 이후로는 직접 만드는 게 훨씬 좋게 느껴져요."

> "이 화면부터는 내가 스스로 스토리를 부여해야 한다고 느껴요."

> "나이도 설정 안 해놨는데 관계성을 설정해야 하잖아요. 생각하기 싫어서 안 하고 싶어요."

> "무에서 유 만드는 건 피로해요."

> "AI가 스토리텔링을 만들어주고, 나는 수정만 하는 건 해볼만 해요."

> "채팅은 좀 기대가 돼요. 성격도 모르고 이미지만 보고 뽑은 건데, 대화해보면 성격을 파악할 수 있고 AI가 어떻게 얘를 만들어놨을지 궁금해서."

## 4. 관계/육성/IP 개입 축 분석

### 4.1 캐스팅만으로 애착 형성: 검증 불가

- 소유감(4점)이 캐스팅 행위가 아닌 예나 직접 생성 경험에서 비롯됨을 본인이 직접 귀인
- "예나 이후로는 직접 만드는 게 훨씬 좋게 느껴져요" — 직접 생성 경험이 기준점을 올림

### 4.2 성장/육성 욕구 언급: 없음

- 육성 관련 발언 없음. 세계관 수정 자체에 피로감 표출

### 4.3 IP 개입 표현: 인식은 있으나 행동 장벽 높음

- 세계관을 제작 행위로 인식(5점)하나 실제 개입은 회피
- "무에서 유 만드는 건 피로해요" — 창작 부담이 개입 장벽으로 작용
- "AI가 스토리텔링 만들어주고 나는 수정만" — AI 초안 제공 시 개입 가능성

## 6. 추가 관찰: 구조 이슈

### 6.1 콘텐츠 포맷 문제

- 텍스트 기반 스토리 소비 의향 낮음. "영상이면 볼 것 같은데" 직접 언급
- 텍스트 소비층과 영상 소비층 분리 설계 필요

### 6.2 세계관 편집 진입 장벽

- 사전 설정(나이 등) 없이 관계성 설정을 요구하는 구조가 피로감 유발
- "나이도 설정 안 해놨는데 관계성을 설정해야 하잖아요" — 선행 설정 없이 관계성 편집 요구하는 구조 문제
- AI가 초안을 먼저 제시하고 수정하는 방식으로 전환 필요

---

## 7. 다음 스프린트 수정 제안 (3개)

1. **세계관 편집 AI 초안 제공**: "무에서 유"가 아닌 "AI 초안 → 사용자 수정" 구조로 전환. "AI가 만들어주고 나는 수정만 하는 건 해볼만 해요" 직접 발언 근거.

2. **스토리 포맷 다변화 검토**: 텍스트 단독 스토리는 이 세그먼트에서 소비 장벽. 짧은 텍스트 + 영상/이미지 조합 또는 자컨 형태 검토.

3. **채팅 기능 우선 노출**: 채팅은 예나 경험과 무관하게 독립적 기대 신호 발생. 세계관 수정 이전 또는 병렬로 채팅 진입 경로 제공 검토.
