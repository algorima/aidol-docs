### 3.2 핵심 행동 패턴

1. **비주얼 집중, 텍스트 회피**: 이미지 생성에는 프롬프트 수정까지 반복하지만, 서사는 "데뷔를 한 번 한 적 있음"으로 끝냄
2. **기존 선호 재확인**: Sprint 1에서 선택한 멤버(라온, 소담, 새별, 소미)를 재선택 -- 선호의 일관성
3. **직접 생성 멤버(예나)가 전체 경험의 앵커**: 소유감, 아쉬움, 채팅 기대 모두 예나 중심
4. **세계관/스토리 콘텐츠에서 즉각 이탈**: 홈 진입 후 하나만 클릭하고 "나갈 것 같다"
5. **"쓰는 것" 자체에 대한 저항**: 총평에서 "쓰는 게 피로"라고 명시

- **패턴**: 비주얼 결과물에 긍정 반응, 텍스트/서사 작업에 부정 반응. 감정의 진폭이 크지 않은 편(전반적으로 절제된 반응)

## 5. Empathy Map

### 5.1 Says (말한 것)

- "맞아요(더 크게)." -- 눈 크기에 대한 수정 요구 (VC-05)
- "실버와 블랙이 섞인 광야 느낌" -- 로고 비전 (VC-07)
- "데뷔를 한 번 한 적 있음. 그리고 끝." -- 서사 최소 입력 (VC-06)

- "어려워요... 불편해요... 생각하기 싫어요." -- 세계관 발전 거부 (VC-10)
- "다른 사람들이 만들어 놓은 멤버들은 별로 안 궁금해요." (VC-15)
- "이미지가 잘생겼으면 눈길이 가겠네요." (VC-15)
- "몰입은 됐는데... 쓰는 게 피로... 즐기기 어려움." (VC-13)

### 5.2 Thinks (생각하는 것)

- 내가 직접 만든 멤버(예나)는 특별하지만, 나머지는 "고른 것"에 가까움
- 비주얼은 중요하고 시간 투자할 가치가 있지만, 텍스트 작성은 귀찮고 불편함
- 세계관이 "내 것"이라는 것은 알겠는데, 그걸 직접 발전시키는 건 원하지 않음
- 텍스트 콘텐츠는 재미가 없고, 영상이어야 볼 만함
- 다른 사람의 결과물에는 관심 없지만, 비주얼이 좋으면 눈길은 감

### 5.3 Does (행동)

- 이미지 프롬프트를 수정하며 외모 디테일에 투자
- 서사는 한 줄로 끝냄
- Sprint 1 동일 멤버를 재선택 (선호 일관성)
- 세계관 홈에서 하나만 클릭 후 즉각 이탈 의사
- 로고 프롬프트에 구체적 미적 비전 제시
- 점수 응답은 명확하게 제공

### 5.4 Feels (느끼는 것)

- **만족**: 이미지/로고가 잘 나왔을 때
- **소유감**: 직접 만든 예나에 대해
- **피로/불편**: 텍스트 입력, 서사 작성, 세계관 구체화 요구 시
- **무관심**: 세계관 스토리 콘텐츠, 타 유저 결과물
- **기대**: 예나와의 채팅 서비스

## 6. Thematic Analysis

### Theme 1: 비주얼 투자 vs 텍스트 회피의 극단적 대비

- 관련 코드: `외모-집착`, `프롬프트-수정반복`, `서사-최소입력`, `텍스트-부담`, `텍스트-거부`, `입력피로`
- 근거: VC-05, VC-06, VC-10, VC-12, VC-13
- 요약: 최지윤은 프로필 이미지에는 "눈이 너무 작다"며 수정을 반복하고, 로고에는 "실버와 블랙이 섞인 광야 느낌"이라는 구체적 비전을 제시한다. 반면 서사는 "데뷔를 한 번 한 적 있음. 끝."이고, 세계관 발전은 "어려워요... 불편해요... 생각하기 싫어요"(1점)이다. 비주얼 창작과 텍스트 창작은 완전히 다른 유형의 행위로 인식되고 있다.

### Theme 2: 직접 생성 멤버가 전체 애착의 앵커

- 관련 코드: `소유감-직접생성`, `예나-앵커`, `소유감-직접생성기반`, `애착-4점`, `아쉬움-4점`
- 근거: VC-01, VC-03, VC-11, VC-14
- 요약: 소유감(4점), 아쉬움(4점), 채팅 기대 모두 "직접 만든 예나" 기반이다. 캐스팅한 멤버(라온, 소담, 새별, 소미)에 대한 독립적 애착 언급은 없다. 직접 생성 1명이 그룹 전체의 애착을 견인하는 구조.

### Theme 3: "내 것" 인식과 "발전 의지"의 분리

- 관련 코드: `제작인식-5점`, `발전욕구-1점`, `비어있음-인지`, `설정피로`, `제작인식-소유감-분리`
- 근거: VC-09, VC-10
- 요약: 제작 인식 5점("누가 다 정해준 게 아니라 비어있는 느낌")이지만 발전 욕구 1점. 이것은 "내가 만들어야 한다는 것은 안다. 하지만 그 작업을 하고 싶지는 않다"는 의미이다. 소유권 인식과 제작 행위 의지는 독립적인 변수임을 보여준다.

### Theme 4: 콘텐츠 포맷 미스매치 -- 텍스트/채팅 vs 영상

- 관련 코드: `텍스트-거부-2점`, `영상-기대`, `콘텐츠포맷-미스매치`, `다음이야기-무관심`
- 근거: VC-12
- 요약: "다음 이야기 궁금" 2점. 영상이 아니면 실망. 하이라이트 콘텐츠가 텍스트/채팅 형식인 것 자체가 관심 부재의 원인이다. 콘텐츠의 내용(세계관 스토리)보다 포맷(텍스트 vs 영상)이 더 큰 결정 요인.

### Theme 5: 소셜/커뮤니티 동기 부재

- 관련 코드: `UGC-무관심`, `소셜-약함`, `비주얼-유일한트리거`
- 근거: VC-15
- 요약: "다른 사람들이 만들어 놓은 멤버들은 별로 안 궁금해요." 타 유저의 결과물에 관심이 없으며, 유일한 예외는 "이미지가 잘생겼으면 눈길이 가겠네요"라는 비주얼 트리거뿐이다. 경쟁/선점/커뮤니티 동기가 약하고, 개인적 소유 경험에 집중된 유저.

## 10. Mental Model Gap Analysis

### Gap 1: 세계관 = 소유 인식 도구 vs 세계관 = 텍스트 작업 부담

| 서비스 모델 | 사용자 멘탈 모델 |
|------------|----------------|
| 세계관 수정/구체화 -> 애착 강화 (H4 가설) | 세계관은 "내 것"이라는 표식일 뿐 (5점). 구체화는 부담 (1점) |
| 세계관 수정 = 제작 행위 -> 소유감 | 수정 행위 자체를 원하지 않음 -> 소유감은 이미 생성 시점에 형성됨 |

**영향**: 세계관 수정이 애착을 강화한다는 H4 가설은 이 유저에게 작동하지 않는다. 소유감은 "만든 순간"에 이미 완성되며, 추가 작업은 오히려 부담.

### Gap 2: 하이라이트 콘텐츠 = 텍스트/채팅 vs 기대 = 영상

| 서비스 모델 | 사용자 멘탈 모델 |
|------------|----------------|
| AI 생성 텍스트/채팅 콘텐츠로 스토리 전달 | "영상이 아니면 실망" -> 텍스트 콘텐츠는 재미없음 |
| 텍스트가 비용 효율적 MVP | MVP여도 흥미를 끌지 못하면 검증 자체가 불가 |

**영향**: 텍스트/채팅 형식의 하이라이트 콘텐츠는 이 유저 유형에게 작동하지 않는다. 최소한 이미지 기반 스토리텔링이나 짧은 영상 클립이 필요.

### Gap 3: UGC/소셜 = 성장 엔진 vs 유저 = 개인 소비자

| 서비스 모델 | 사용자 멘탈 모델 |
|------------|----------------|
| 타 유저 그룹 탐색, 좋아요/관심 -> 애착 증가 | "다른 사람들이 만들어 놓은 멤버들은 별로 안 궁금해요" |
| 소셜 기능이 리텐션 드라이버 | 개인적 소유 경험만으로 충분. 소셜 불필요 |

**영향**: 소셜/커뮤니티 기능은 이 유저 유형에게 리텐션 드라이버가 아니다. 비주얼 품질이 유일한 소셜 트리거("잘생겼으면 눈길").

## 11. Norman's 7 Stages of Action Analysis

### 11.1 Goal (목표)
- **유저 목표**: 내 취향대로 예쁜 아이돌을 만들고, 소유하고, 가볍게 관계하기
- **시스템 목표**: 캐스팅 + 직접 생성 -> 포지션 -> 그룹 완성 -> 세계관 -> 콘텐츠 소비

### 11.2 Execution Side (실행)

| 단계 | 분석 |
|------|------|
| Plan (계획) | Sprint 1 경험 기반으로 어떤 멤버를 고를지 이미 알고 있음 -> 빠른 재선택 |
| Specify (구체화) | 직접 생성 시 외모 디테일 구체화 ("살색 머리, 갸름한 턱, 큰 눈") -> 명확한 비전 |
| Perform (수행) | 캐스팅/직접생성/포지션/그룹명/로고 -> 전체적으로 무난하게 수행. 이미지 수정 1회 발생. |

### 11.3 Evaluation Side (평가)

| 단계 | 분석 |
|------|------|
| Perceive (인지) | 이미지 결과물 즉각 확인 -> "눈이 너무 작아요" (정확한 인지) |
| Interpret (해석) | 수정 후 재생성 결과 -> "예쁘다", 로고 -> "멋있다" (기대 부합) |
| Compare (비교) | 비주얼 결과물: 기대 충족. 세계관/텍스트 콘텐츠: 기대 미달 (포맷 문제) |

### 11.4 Gulf of Execution (실행의 간극)
- 이미지 생성 프롬프트가 의도대로 작동하지 않을 때 수정 필요 (눈 크기) -- 간극 존재하나 1회 수정으로 해소
- 서사 입력이 "필수"로 느껴질 때 부담 -- "데뷔 한 번 한 적 있음"으로 최소 우회

### 11.5 Gulf of Evaluation (평가의 간극)
- 세계관 홈에서 "뭘 해야 하는지" 모호 -> 하나 클릭 후 이탈
- 하이라이트 콘텐츠가 텍스트 -> "이게 콘텐츠인가?" 수준의 기대 괴리

## 12. Norman's 3 Levels of Design

### 12.1 Visceral Level (본능적 반응)

- **긍정**: 직접 생성 이미지 수정 후 결과 ("예쁘다"), 로고 ("와 멋있다")
- **부정**: 이미지 생성 첫 결과 ("눈이 너무 작아요")
- **시사점**: Visceral 레벨에서 비주얼 품질이 곧 만족도. 첫 생성 결과의 품질이 높을수록 마찰 감소.

### 12.2 Behavioral Level (행동적 반응)

- **긍정**: 캐스팅 플로우 자체는 빠르고 무난하게 수행 (재방문이라 학습 효과)
- **부정**: 서사 입력 단계에서 최소 투자 ("끝"), 세계관 홈에서 무엇을 해야 하는지 불명확
- **시사점**: 텍스트 입력이 필수인 단계마다 마찰 발생. 입력을 선택사항으로 만들거나 비주얼 기반 대안 필요.

### 12.3 Reflective Level (반성적 반응)

- **긍정**: "내 것"이라는 소유 인식 형성 (제작 인식 5점), 예나에 대한 소유감/아쉬움 (4점)
- **부정**: "쓰는 게 피로... 즐기기 어려움" -> 서비스 전체에 대한 Reflective 평가가 부담으로 귀결
- **시사점**: 소유감은 형성되었으나, 서비스 지속 사용의 Reflective 동기가 약함. "내 것"을 만들었다는 만족 이후 "이제 뭘 하지?"에 대한 답이 필요.

## 15. Cross-Analysis Summary

### 15.1 핵심 인사이트 5선

**1. "소유감"과 "발전 의지"는 독립 변수다 (제작 인식 5점 / 발전 욕구 1점)**

이 유저에게 소유감은 "만든 순간" 완성된다. 세계관을 추가로 구체화해야 소유감이 강화된다는 H4 가설은 작동하지 않는다. 오히려 세계관 수정 요구는 부담으로 작용하여 이탈 리스크를 높인다.

액션: 세계관 구체화를 "필수 단계"가 아닌 "선택적 확장"으로 설계. 생성 완료 시점에서 이미 "내 것"이라는 인식을 강화하는 피드백(축하 메시지, 소유권 뱃지 등)이 더 효과적.

**2. 직접 생성 멤버 1명이 그룹 전체 애착의 앵커**

캐스팅 멤버 4명(라온, 소담, 새별, 소미)에 대한 독립적 애착 언급은 전무하다. 모든 소유감/아쉬움/채팅 기대가 직접 생성한 예나에 집중. Sprint 1의 H1("캐스팅만으로 애착")은 이 유저에게 지지되지 않으며, 직접 생성이 애착의 필수 조건이다.

액션: "직접 생성" 기능을 캐스팅 플로우의 핵심 단계로 승격. 최소 1명은 직접 만들도록 유도하는 플로우 설계.

**3. 비주얼 투자 vs 텍스트 회피의 극단적 분리**

이미지 생성에는 프롬프트 수정까지 반복하고 로고에 구체적 비전을 제시하지만, 서사는 한 줄, 세계관 발전은 1점이다. "몰입은 됐는데 쓰는 게 피로"라는 총평이 이 유저의 핵심 UX 병목을 요약한다.

액션: 텍스트 입력 의존도 최소화. 서사/세계관은 선택지 기반(객관식, 카드 선택) 또는 비주얼 기반(무드보드, 이미지 선택)으로 대체.

**4. 텍스트 콘텐츠 포맷은 이 유저 유형에게 리텐션 실패 요인**

"다음 이야기 궁금" 2점, "영상이 아니면 실망". 하이라이트 콘텐츠가 텍스트/채팅 형식인 것 자체가 흥미 부재의 원인. 채팅 서비스는 기대하지만("해볼 것 같아요"), 이는 "예나와의 관계"에 대한 기대이지 텍스트 포맷에 대한 선호가 아님.

액션: 하이라이트 콘텐츠에 이미지/짧은 영상 클립 우선 적용. 텍스트만의 스토리 전달은 이 세그먼트에게 무효.

**5. 소셜/커뮤니티 동기는 보편적이지 않다**

U1/U2(은찬/예찬)에서 강하게 나타난 경쟁/선점/바이럴 동기가 U10(최지윤)에게는 거의 없다. "다른 사람들이 만들어 놓은 멤버들은 별로 안 궁금해요." 유일한 소셜 트리거는 비주얼 품질("잘생겼으면 눈길").

액션: 소셜 기능을 핵심 리텐션 메커니즘으로 전제하지 말 것. 개인적 소유 경험이 충분한 가치를 제공하도록 설계하되, 소셜은 선택적 레이어로.
