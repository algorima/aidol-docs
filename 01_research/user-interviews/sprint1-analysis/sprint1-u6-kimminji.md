# SPRINT 1 - 캐스팅 인식 검증: U6 김민지

---

## 1. 세그먼트 정보

- 28세 / 여성 / 디자이너
- 덕후 레벨 5/10 (오덕 수준)
- 데이식스 팬
- 플레이브 인지
- AI 채팅 경험은 실무 보조용 (오류, 문장 정리 등)
- 버추얼 아이돌 소비 경험은 깊지 않음

**중간 레벨 팬 + 실력 중심 소비 성향**

---

## 2. UT 설계 목적별 분석

---

### 2.1 캐스팅을 '선택 행위'로 인지하는가?

#### 관찰 데이터

- "PD 역할을 했다."
- "내가 키우는 내 아이라는 생각이 들었다."
- 보이그룹 직접 선택
- 능력치 중심 판단
- MBTI는 참고만, 핵심은 능력치
- 얼굴 + 실력 밸런스 고려

#### 해석

- 단순 클릭으로 인지하지 않음
- 명확히 '기획/육성 행위'로 인식
- 전략 + 감정 결합된 상태

**H1 (선택 행위 인지) 강하게 검증**

---

### 2.2 캐스팅만으로 애착이 발생하는가?

#### 관찰 데이터

- "내가 키우는 내 아이"
- 공유 의향 있음
- 홍보 의지 발화
- 성공 전략 사고 (라디오 돌리기 등)
- 경연, 스탯 성장 기대

#### 해석

- 전략형 재미를 넘어 **육성 애착 초기 단계 형성**
- '내 아이' 표현은 정서적 동일시 신호
- 다만 깊은 집착/독점 발화는 없음

**H1 애착 형성 부분 검증 (차유경보다 강함)**

---

### 2.3 이메일 입력은 기대 기반 전환인가?

직접 이메일 질문 데이터는 없지만:
- 경연/스탯 성장 기대
- 장기 육성 관점 존재
- '내가 홍보한다' 사고

**장기 서사형 서비스라면 전환 가능성 있음** (연습생 단발 콘텐츠로는 약함)

---

## 3. UT 구조별 관찰 정리

---

### 3.1 AI 연습생 캐스팅

- 가능한 멤버 우선 선택
- 직접 생성 시도했으나 속도/완성도 문제로 이탈
- 얼굴 퀄리티 불만
- 능력치 재확인 UI 필요성 제기

---

### 3.2 그룹 콘셉트 설정

- 보이그룹 선택
- 실력/무대 중심 사고
- 케미보다 실력/칼각 강조

---

### 3.3 결과 카드 확인

- 공유 의향 있음
- 홍보 전략 사고 존재
- 육성 관점 기대

---

### 3.4 이메일 입력

- 직접 데이터 없음
- 장기 육성형 구조면 긍정적 가능성

---

## 4. 정량 지표 관점 해석 (정성 기반)

| 지표 | 관찰 |
| --- | --- |
| 그룹 생성 완료 | 완료 |
| 캐스팅 고민 | 존재 |
| 특정 멤버 동일시 | "내 아이" 발화 |
| 결과물 공유 의향 | 있음 |
| 장기 기대 신호 | 존재 (경연/육성 기대) |

---

## 5. 가설 검증 요약 (김민지 단독)

---

### H1: 유저는 캐스팅만으로 애착을 느낀다

부분~중간 강도 검증

- 선택 행위 인지 (O)
- 동일시 발화 (O)
- 전략적 육성 사고 (O)
- 집착/독점 감정은 아직 약함

---

### H2: 결과물보다 선택 과정에 의미를 둔다

부분 검증

- 과정은 게임처럼 즐김
- 하지만 '성장 이후 결과'에 더 관심 있음
- 즉, 선택 자체보다 **육성 서사**가 핵심

---

### H3: 이메일 입력은 기대 행동이다

미검증 (직접 질문 없음)

- 단발 콘텐츠라면 전환 약함
- 육성형 서사라면 가능성 있음

---

## 김민지 인터뷰 핵심 요약

1. 캐스팅은 명확히 PD 행위로 인지됨.
2. '내 아이' 발화 -> 초기 애착 형성.
3. 실력 중심 팬 -> 육성/경연 시스템에 강한 반응 예상.
4. 단발 그룹 생성보다는 성장 시스템이 있어야 몰입.
5. 얼굴 퀄리티 + 능력치 재확인 UI가 몰입 핵심 변수.
