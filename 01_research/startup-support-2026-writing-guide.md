| 항목 | 초기창업패키지 (초창패) | 창업성공패키지 청년창업사관학교 (청창사) |

## 2. 공통 평가 프레임워크: PSST

두 사업 모두 동일한 4대 평가축을 사용한다.

| 평가축 | 초창패 섹션명 | 청창사 섹션명 | 핵심 질문 |
|---|---|---|---|
| **P** (Problem) | 1. 문제 인식_창업 아이템의 필요성 | 1. 문제인식 (1-1 개발동기, 1-2 목적/필요성) | 이 문제가 진짜 존재하는가? 얼마나 큰가? |
| **S** (Solution) | 2. 실현 가능성_창업 아이템의 개발 계획 | 2. 실현가능성 (2-1 개발방안, 2-2 대응방안) | 사업기간 내 실현 가능한가? 차별점이 있는가? |
| **S** (Scale-up) | 3. 성장전략_사업화 추진 전략 | 3. 성장전략 | 수익모델은? 시장 진입/확장 전략은? |
| **T** (Team) | 4. 팀 구성_대표자 및 팀원 구성 계획 | 4. 팀 구성 | 이 팀이 해낼 수 있는가? |

## 3. 평가항목별 상세 기준

### 3.1 초기창업패키지 평가 기준

**서류평가**

- 2배수 선발 (발표평가 대상자 결정)
- 평가 소요: 건당 15~20분
- 60점 미만 탈락

**서류평가 가점 (최대 3점)**

| 가점 항목 | 배점 |
|---|---|
| 인공지능 대학원 졸업생 | 2점 |
| 전국 규모 창업경진대회 장관급 이상 수상 | 1점 |
| 기후테크 분야 | 1점 |
| 최근 1년 내 1억원 이상 투자유치 | 1점 |
| 1인 창조기업센터 추천 입주기업 | 1점 |

**서류평가 면제 대상**

- CES 혁신상 수상 기업
- 도전! K-스타트업 왕중왕전 진출 기업

- 성실경영 심층평가 통과 청년기업
- 예비/재도전 성공패키지 '최우수' 판정 기업

**심층인터뷰 (2월 말)**

- 30분, 탈락자 없음
- 사업계획서 내용 기반 질의응답
- 대표자의 이해도와 진정성 확인

**발표평가 (3월)**

- 발표 20분 + Q&A 10분
- 평가 결과에 따라 사업비 차등 지급 (높은 점수 = 더 많은 사업비)

**발표평가 우대**

- 24년 도전! K-스타트업 왕중왕전 수상기업
- CES 최고혁신상 수상 기업
- D-DAY/정주영 창업경진대회 입상 후 추천 기업

## 4. PSST 섹션별 작성 전략

### 4.1 P (Problem): 문제 인식

**평가위원이 보는 것:** "이 문제가 진짜 돈이 되는 문제인가?"

**필수 요소**

1. 거시적 환경 변화 (PEST 분석 활용)
   - 정치/규제, 경제, 사회/문화, 기술 트렌드 중 관련 있는 것
2. 시장 규모의 정량적 근거
   - 출처 명시 (통계청, 산업 리포트, 학술 논문 등)
   - TAM → SAM → SOM 구조
3. 타겟 고객의 구체적 페인포인트
   - 고객 인터뷰, 설문, 리뷰 분석 등 1차 데이터
4. 기존 솔루션의 한계
   - 왜 현재 대안이 부족한지

**금지 사항**

- "많은 사람들이 불편해한다" 같은 주관적 표현
- "국내 최초", "경쟁사 없음" (오히려 감점 요인)
- 출처 없는 시장 규모 (예: "40조원 시장")
- 개인적 느낌이나 추측

**AIoIA 적용 포인트**

- K-pop 팬덤 222개국 3,252만명 (글로벌 데이터)
- 팬-아이돌 인터랙션 불가능 비율 90%+ (구조적 문제)
- AI Companion 시장 $37.7B (CAGR 31.2%)
- Character.AI 1억명 사용자 = 시장 검증 완료

### 4.2 S (Solution): 실현 가능성

**평가위원이 보는 것:** "사업기간 내에 진짜 만들 수 있는가?"

**필수 요소**

1. 현재까지의 성과/진행 상황 (가장 중요)
   - 프로토타입, MVP, 베타 버전
   - 사용자 데이터 (DAU, 리텐션, 전환율)
   - TikTok 바이럴 실적 등 트랙션
2. 제품/서비스 구체적 개발 계획
   - 기능 스펙, 기술 스택
   - 차별성과 경쟁력 확보 전략
3. 사업추진 일정 (간트차트 형식 권장)
   - 언제까지, 무엇을, 어떻게
   - 마일스톤과 산출물 명시
4. 사업비 집행 계획
   - 비목별 상세 내역
   - 각 항목의 논리적 근거 (왜 이 금액인지)

**금지 사항**

- 기술 설명만 나열하고 실행 계획 없음
- 사업비 배정 근거 없이 숫자만 기입
- "추후 결정" 같은 모호한 계획

**AIoIA 적용 포인트**

- MVP 완성도: Next.js + FastAPI + GPT 기반
- TikTok 검증: $25 → 500K views, 5,000 리드 (CAC $0.005)
- 워크샵 전환율 30% ($150 티켓)
- 기술 스택 명시: Fish Audio TTS, LiveKit, GPT Image

### 4.3 S (Scale-up): 성장전략

**평가위원이 보는 것:** "어떻게 돈을 벌고 키울 것인가?"

**필수 요소**

1. 경쟁사 분석
   - 기능 비교표보다 "우리만의 접근법" 강조
   - "기존 솔루션의 한계를 A방식으로 해결한 첫 번째 접근"
2. 비즈니스 모델 (수익화 전략)
   - 구체적 가격 포인트, 과금 구조
   - 단위 경제학 (CAC, ARPU, LTV)
3. 시장 진입 전략
   - 진입 단계 vs 성장 단계 구분
   - 채널별 구체적 마케팅 계획과 예산
4. 투자유치/자금확보 전략
5. 사업 전체 로드맵
6. ESG 계획 (청창사 특히 중요)
   - 환경: 탄소 절감, 친환경 등
   - 사회: 지역사회, 다양성 등
   - 지배구조: 윤리경영, 투명 경영

**금지 사항**

- "인스타그램 운영하겠다" (구체성 0)
- 마케팅 채널만 나열하고 예산/기대효과 없음
- "3년 내 100억 매출" (근거 없는 장밋빛 전망)

**AIoIA 적용 포인트**

- Phase 1: 구독 $3.99/월 → Phase 2: Creator Economy 70/30 → Phase 3: B2B API $10K-50K/월
- 단위 경제학: CAC <$5, ARPU $60-100/year, LTV $150+, LTV/CAC >30x
- Chicken-and-Egg 해결: Alora 프리셋 → UGC 선순환 구조
- 글로벌 진출: US 1순위 → SEA 2순위 (90% 해외 사용자)

### 4.4 T (Team): 팀 구성

**평가위원이 보는 것:** "이 팀이 해낼 수 있는가?"

**필수 요소**

1. 대표자 역량
   - Founder-Market Fit: 왜 내가 이 문제를 풀어야 하는가
   - 관련 경력, 기술력, 노하우
   - 유사 경험, 수행 이력, 수상 실적
2. 팀 구성
   - 이상적: 개발(만드는) + 판매(파는) + 운영(관리) 3축
   - 판매 역량은 반드시 내재화
3. 협력 기관/파트너
   - 구체적 협업 내용과 시기
4. 채용 계획
   - 협약기간 내 채용 예정 인력

**금지 사항**

- 학교명만 기재하고 실제 역량 미기술
- "다양한 경험 보유" 같은 추상적 표현
- 팀원 기술력 부족을 인정하지 않고 넘어감

**AIoIA 적용 포인트**

- 대표 소연: HCI/UX 전문성 + 20년 K-pop 팬덤 경험 (Founder-Market Fit)
- CTO: KAIST 출신, ScatterLab에서 이루다 2.0 개발 경험
- 3번의 피벗 경험 = 학습 과정 투명 공개 (AI Boyfriend → BTS AI → AI Idol Platform)

## 5. 서류심사 통과 핵심 전략

평가위원은 서류평가에서 건당 15~20분 내에 판단한다. 핵심은 빠르게 읽히는 구조.

### 5.1 문서 구성 원칙

1. **첫 페이지(개요)에서 승부** - 개요표에서 Problem → Solution → Scale-up → Team의 핵심을 한 줄씩 강렬하게
2. **표와 시각자료 적극 활용** - 텍스트 나열 대신 도표, 그래프, 제품 이미지
3. **각 섹션 첫 문장이 결론** - 두괄식 작성. 근거는 뒤에 배치
4. **정량 데이터 볼드 처리** - 숫자는 눈에 띄게

### 5.2 흔한 탈락 사유

| 탈락 사유 | 대응 전략 |
|---|---|
| 문제 정의가 모호함 | 데이터 기반, 구체적 고객 세그먼트 |
| 기술 나열만 있고 실행 계획 없음 | 마일스톤별 일정 + 산출물 명시 |
| 경쟁사 분석이 "경쟁사 없음" | 유사 서비스 3개 이상 비교분석 |
| 사업비 근거 부족 | 비목별 산출 근거 기재 |
| 팀 역량 불분명 | 관련 경력 구체적 기술 |

## 6. 발표심사 통과 핵심 전략

### 6.1 발표 구성 (초창패 20분 / 청창사 15분)

| 시간 | 내용 | 비중 |
|---|---|---|
| 1~2분 | Hook: 핵심 문제와 우리의 해결책 (한 문장) | 10% |
| 3~5분 | Problem: 시장의 고통 + 크기 | 20% |
| 5~8분 | Solution: 제품 데모 + 트랙션 | 30% |
| 3~5분 | Scale-up: BM + 성장 전략 | 25% |
| 2~3분 | Team + Ask | 15% |

### 6.2 Q&A 대비 예상 질문

| 영역 | 예상 질문 |
|---|---|
| 기술성 | "자체 기술과 외부 API 비중은?" "기술적 진입장벽은?" |
| 사업성 | "첫 매출 시점은?" "BEP 달성 계획은?" |
| 경쟁 | "Character.AI와 뭐가 다른가?" "네이버 캐릭터챗과의 차이는?" |
| 시장 | "왜 K-pop인가?" "시장이 충분히 큰가?" |
| 팀 | "핵심 인력이 빠지면?" "기술 인력 충원 계획은?" |
| 사업비 | "이 금액의 산출 근거는?" "자부담 현금은 어떻게 마련?" |

## 7. 초창패 vs 청창사 차별화 전략

같은 AIoIA 사업이지만 두 사업의 강조점을 다르게 가져가야 한다.

### 7.1 초창패에서 강조할 것

- **기술의 구체성**: 자체 개발 AI 페르소나 엔진, 기술 스택 상세
- **트랙션 데이터**: TikTok CAC $0.005, 500K views, 5,000 리드
- **Phase 1 → Phase 2 전환 로직**: 개인 AI → UGC 플랫폼 확장 시나리오
- **사업비 집행의 논리성**: 비목별 상세 산출근거

### 7.2 청창사에서 강조할 것

- **대표자의 기업가정신**: 20년 팬덤 경험, 3번의 피벗, Founder-Market Fit
- **글로벌 확장 가능성**: 90% 해외 사용자, YC 지원 경험, US/SEA 진출 전략
- **중점 모집분야 매칭**: 초격차/신산업 분야 (AI Entertainment)
- **ESG 도입 계획**: K-pop이 가진 문화 다양성, 글로벌 커뮤니티 가치
- **5T 프로그램 활용 계획**: 청창사의 특화 프로그램과 연계

## 8. 체크리스트

### 8.1 제출 전 최종 점검

- [ ] 문제 정의에 정량적 데이터가 3개 이상 있는가
- [ ] 시장 규모에 출처가 명시되어 있는가 (TAM/SAM/SOM)
- [ ] 경쟁사를 3개 이상 분석했는가
- [ ] "우리만의 차별점"이 한 문장으로 정리되는가
- [ ] 사업추진 일정이 월 단위로 구체적인가
- [ ] 사업비 비목별 산출 근거가 있는가
- [ ] 팀원의 역량이 "이 사업"과 연결되어 기술되었는가
- [ ] 대표자의 Founder-Market Fit이 드러나는가
- [ ] 페이지 수 제한을 준수했는가 (초창패 15p / 청창사 10p)
- [ ] 개인정보가 마스킹 되었는가

### 8.2 D2SF 지원서에서 배운 교훈 (자체 회고)

| 교훈 | 적용 |
|---|---|
| "API 조합"으로 읽히면 안 됨 | 자체 기술 요소를 전면에 배치 |
| 경쟁사 비교가 역효과 날 수 있음 | "한계 → 우리의 접근" 프레이밍 |
| 일방향 수혜 프레이밍 금지 | "상호 가치" 관점으로 서술 |
| 팀원 이력 빈약 = 기술력 의심 | 구체적 경력/성과 중심 기술 |
| 리텐션 데이터 약점 직시 | 약점은 인정 + 개선 계획 제시 |
