# 비즈니스 모델 (Business Model)

## 수익 모델

K-pop 팬덤 경제 모델 기반, 다중 수익원 전략

### 1. 구독 (~$5/월/아이돌)
- 독점 1:1 메시지
- 프리미엄 콘텐츠
- **벤치마크**: Bubble 230만+ 유료 구독자

### 2. 프리미엄 멤버십 (~$50/년)
- 디지털 굿즈
- 온라인 이벤트 우선 참여권
- 공식 팬클럽 자격

### 3. 광고 & 브랜드 파트너십
- AI 아이돌 인기도 기반 브랜드 딜
- 크리에이터 경제 모델

### 4. (Phase 2+) 플랫폼 수수료
- 크리에이터가 AI 아이돌 제작 → 플랫폼에서 팬덤 형성
- 수익의 일부 수수료
- **비전**: "AI 동반자를 위한 YouTube"

## 단위 경제 (Unit Economics) 예상

| 지표 | 목표 |
|------|------|
| CAC | < $5 (바이럴 + 오가닉) |
| ARPU | $60~100/년 |
| LTV | $150+ (3년 기준) |
| LTV/CAC | > 30x |

## 수익화 로드맵

```
Phase 1 (MVP): 무료 사용 → 사용자 확보 & PMF 검증
     ↓
Phase 2: 구독 모델 도입 → 핵심 팬 수익화
     ↓
Phase 3: 프리미엄 멤버십 + 이벤트 → ARPU 상승
     ↓
Phase 4: 플랫폼화 → 크리에이터 생태계
```

## 비용 구조

| 항목 | 비중 | 설명 |
|------|------|------|
| AI API 비용 | 30-40% | GPT, 이미지 생성, 음성 합성 |
| 인건비 | 30-40% | 개발, 운영, 콘텐츠 |
| 마케팅 | 15-20% | TikTok, 인플루언서 |
| 인프라 | 5-10% | 클라우드, 서버 |
