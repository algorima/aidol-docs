# 비즈니스 모델 (Business Model)

## 수익 모델

시즌 기반 시뮬레이션 게임의 다중 과금 구조

### Phase 2 게임 기반 과금

| 과금 항목 | 가격대 | 설명 |
|-----------|--------|------|
| **시즌 패스** | 월 3,000~10,000원 | 시즌별 프리미엄 콘텐츠, 추가 이벤트, 독점 연습생 풀 접근 |
| **추가 드래프트 슬롯** | 회당 1,000~3,000원 | 기본 그룹 외 추가 그룹 생성/캐스팅 권한 |
| **프리미엄 콘셉트 팩** | 팩당 2,000~5,000원 | 특별 콘셉트, 한정 의상, 레어 연습생 |
| **시뮬레이션 확장** | 월 구독 | 추가 대화 턴, 심화 관계 시뮬레이션, 시즌 이벤트 |

### Phase 3+ 확장 과금

| 과금 항목 | 설명 |
|-----------|------|
| 크리에이터 마켓 수수료 | 유저가 만든 콘셉트/연습생을 거래할 때 플랫폼 수수료 |
| 디지털 굿즈 | AI 아이돌 기반 포토카드, 앨범 등 |
| 브랜드 파트너십 | AI 아이돌 인기도 기반 브랜드 딜 |

## Growth Flywheel

```
유저 증가
  → 드래프트 경쟁 강도 상승 (인기 연습생 조기 선점)
    → 선점 가치 증가 (빨리 참여해야 좋은 연습생을 뽑을 수 있다)
      → 유료 전환 증가 (추가 슬롯, 프리미엄 풀)
        → ARPU 상승
          → IP 다양성 증가 (더 많은 유저 그룹 = 더 풍부한 콘텐츠)
            → 플랫폼 매력 증가
              → 유저 증가 (반복)
```

핵심: 유저가 많아질수록 드래프트가 경쟁적이 되고, 경쟁이 치열해질수록 유료 전환이 올라가는 구조.

## 단위 경제 (Unit Economics)

| 지표 | 목표 |
|------|------|
| CAC | < $5 (바이럴 + 오가닉) |
| ARPU | $60~100/년 |
| LTV | $150+ (3년 기준) |
| LTV/CAC | > 30x |

### CAC 근거
- TikTok 바이럴 실험: 500K 조회 → 500 가입 (CAC $0.005)
- 마케팅 예산 $0 실험에서 19명이 자발적 홍보 콘텐츠 제작
- K-pop 팬덤의 자발적 공유 문화 → 오가닉 확산

## 수익화 로드맵

```
Phase 1 (MVP): 무료 사용 → 사용자 확보 & PMF 검증
     |
Phase 2: 시즌 패스 + 드래프트 슬롯 도입 → 핵심 유저 수익화
     |
Phase 3: 프리미엄 콘셉트 + 시뮬레이션 확장 → ARPU 상승
     |
Phase 4: 크리에이터 마켓 + 디지털 굿즈 → 플랫폼 생태계
```
