# aioia.ai Product Hunt 출시 실행 계획 (라이트 버전)

> PH 상위권은 개발자 도구가 독점 (2026년 3월 기준 Top 10 중 8개가 dev tool).
> K-pop 팬 타겟인 AIdol의 **메인 유입 채널은 틱톡/릴스**이며,
> PH는 **"Featured on PH" 뱃지 + AI 검색 인용(GEO) 부스트** 목적으로 가볍게 실행.

## 0. 팀 역할 배정

| 담당자 | PH 역할 | 비고 |
|--------|---------|------|
| 소연 | PH 총괄 (페이지 세팅 + 메이커 코멘트 + 댓글 응답) | 핵심 담당 |
| 영욱 | 공동 메이커 + 기술 질문 대응 + 사이트 최적화 | 핵심 담당 |
| 제이 | PH 썸네일/갤러리 이미지 + **틱톡/릴스 숏폼 (메인)** | 숏폼이 주 업무 |
| 미진 | **퍼포먼스 숏폼 영상 (메인)** | 숏폼이 주 업무 |
| 수지 | 서포터 50명 관리 + Reddit/Discord 포스팅 | |
| 나머지 팀원 | 출시일 댓글 응답 + 소셜 확산 | 출시일만 참여 |

### 1.1 소연

| 할 일 | 산출물 |

| 출시 날짜 확정 + 팀 공유 | **목요일 추천** (1위 기준 400~500 업보트, 뉴스레터 포함) |

### 1.2 영욱

| 할 일 | 산출물 |
|-------|--------|
| aioia.ai FCP 측정 + 0.4초 이하 최적화 | Lighthouse 리포트 |

| PostHog utm_source=producthunt 트래킹 설정 | 전환 추적 |

### 1.3 제이

| 할 일 | 산출물 |
|-------|--------|
| PH 썸네일 1개 | 512x512px PNG |
| 갤러리 이미지 4장 | 1270x760px PNG |
| 데모 영상 1개 | 30초 MP4 (자막 포함) |
| **틱톡 숏폼 4개 사전 촬영** (스피드런, Before/After, Chat Reaction, POV) | 각 15~30초 |

데모 영상 구성 (30초):
- 0~5초: 훅 ("What if you could produce your own K-pop group?")
- 5~15초: 그룹 생성 → 멤버 드래프트 화면
- 15~25초: 채팅 하이라이트 + 멤버 케미
- 25~30초: 로고 + CTA ("Try it free at aioia.ai")

갤러리 이미지 구성:
1. 히어로샷 - 완성된 그룹 + 대화 화면
2. 드래프트/캐스팅 시스템 (게임성)
3. 음성 통화 기능 (차별점)
4. 유저 후기 or 다양한 그룹 예시

## 5. 핵심 원칙

1. **PH에 올인하지 않는다** — 2주 준비, 소연+영욱 중심. 나머지 팀은 숏폼에 집중
2. **틱톡 동시 드롭** — PH 출시 당일 16:15 KST에 스피드런 영상 틱톡 드롭. 두 채널 시너지
3. **서포터는 50명이면 충분** — 100명 모으느라 시간 쓰지 않기. 기존 PH 계정 보유자 우선
4. **"업보트 해줘" 절대 금지** — "피드백 부탁드려요" 방향. PH 알고리즘 조작 감지 주의
5. **진짜 목적을 잊지 않기** — "Featured on PH" 뱃지 + GEO 부스트. 유저 확보는 틱톡에서
