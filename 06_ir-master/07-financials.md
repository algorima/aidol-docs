## 손익분기점 (BEP)

| 지표 | 수치 |
|------|------|
| 월 고정비 (급여 포함) | ~817만원 |
| 복합 ARPU | $10/월 (약 13,000원) |
| BEP 유료 사용자 | **~630명** |
| 전체 사용자 (유료 전환 15% 가정) | ~4,200명 |

### BEP 근거
- 월 고정비 817만원 / 인당 월 과금 13,000원 = 약 630명의 유료 사용자
- 유료 전환율 15% 가정 시, 전체 사용자 약 4,200명에서 BEP 달성
- AI API 비용은 사용자 증가에 비례하여 상승하나, 과금 수익이 이를 상쇄
