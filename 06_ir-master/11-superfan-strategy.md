# 슈퍼팬 전략 (Superfan Strategy)

> 김민철님 피드백: "슈퍼팬들을 어떻게 잘 공략할 것인가에 대한 대비책"

## 1. 슈퍼팬 정의

### 1.1 슈퍼팬이란

상위 10%의 크리에이터로, 플랫폼 매출의 50% 이상을 담당하는 핵심 사용자.
지출 상한이 없으며, 감정적 투자가 금전적 투자로 직결되는 집단입니다.

| 일반 사용자 | 슈퍼팬 |
|-------------|--------|
| 무료 또는 소액 과금 | 월 5만원+ 지출 |
| 그룹 1개 보유 | 그룹 3개+ 보유 |
| 주 1-2회 접속 | 매일 접속 |
| 기본 의상 사용 | 모든 시즌 의상 구매 |
| 조용히 사용 | SNS 공유, 커뮤니티 활동 |

### 1.2 K-pop 팬덤의 슈퍼팬 지출 레퍼런스

| 지출 항목 | 연간 금액 | 출처 |
|-----------|----------|------|
| 미국 슈퍼팬 평균 | **$1,422** (약 185만원) | Luminate 2023 |
| 한국 상위 팬 | **연 100만원+** | 자체 조사 |
| 영통팬싸 1회 | **200~400만원** | 중고나라/번개장터 거래가 |
| BTS 콘서트 | **20~50만원** (국내), **100만원+** (해외) | 티켓+교통+숙박 |
| 포토카드 수집 | **연 50~100만원** | 팬 커뮤니티 |

AIoIA에서의 슈퍼팬 예상 지출: **월 5~15만원** (의상 전종 + 프리미엄 콘셉트 + 시즌패스 + 추가 멤버)
연간 기준: **60~180만원** — K-pop 슈퍼팬 지출 범위 내

## 2. 슈퍼팬 식별 지표

| 지표 | 측정 방법 | 슈퍼팬 기준 |
|------|-----------|-------------|
| 생성 빈도 | 그룹/멤버 생성 수 | 월 3개+ 그룹 |
| 과금 빈도 | 유료 아이템 구매 횟수 | 월 5회+ 구매 |
| 체류 시간 | 일 평균 세션 시간 | 일 30분+ |
| 공유 횟수 | SNS 공유, URL 공유 수 | 주 3회+ |
| 채팅 활동 | 일 평균 채팅 턴 수 | 일 50턴+ |
| 시즌 완주율 | 시즌 콘텐츠 완료율 | 매 시즌 100% |

### 슈퍼팬 스코어 산출

```
슈퍼팬 스코어 = (생성 빈도 x 2) + (과금 빈도 x 3) + (체류 시간 x 1)
              + (공유 횟수 x 2) + (채팅 활동 x 1) + (시즌 완주 x 2)
```

상위 10% → 슈퍼팬 등급 자동 부여

## 3. 프리미엄 혜택 설계

### 3.1 탐색 페이지 상단 노출 (김민철님 직접 제안)

| 항목 | 내용 |
|------|------|
| 기능 | 슈퍼팬의 그룹이 탐색 피드 상단에 노출 |
| 조건 | 슈퍼팬 등급 달성 시 자동 |
| 효과 | 노출 증가 → 팔로워 증가 → "더 키우고 싶다" 동기 강화 |
| 비유 | 인스타그램 Explore 추천 알고리즘과 유사 |

이 혜택은 슈퍼팬에게 가장 강력한 동기를 줍니다:
- 금전적 보상이 아닌 **인정/노출** 보상
- "내 그룹이 많은 사람에게 보여진다" = 크리에이터의 핵심 욕구

### 3.2 독점 콘텐츠

| 혜택 | 설명 |
|------|------|
| 독점 콘셉트 | 슈퍼팬만 구매 가능한 한정 콘셉트 |
| 독점 의상 | 시즌 한정 의상 (일반에게 미공개) |
| 히든 시즌 | 슈퍼팬 전용 사이드 스토리/미니 시즌 |

### 3.3 얼리 액세스

| 혜택 | 설명 |
|------|------|
| 신규 기능 | 새 기능 1주 먼저 체험 |
| 시즌 프리뷰 | 다음 시즌 콘셉트 미리 공개 |
| 베타 테스트 | 신규 시스템 테스트 참여 |

### 3.4 크리에이터 뱃지 / 랭킹

| 뱃지 | 조건 | 효과 |
|------|------|------|
| 골드 크리에이터 | 슈퍼팬 스코어 상위 10% | 프로필에 뱃지 표시 |
| 시즌 챔피언 | 시즌별 인기 투표 1위 | 시즌 트로피 + 영구 기록 |
| 레전드 | 3시즌 연속 챔피언 | 전용 프레임 + 홀 오브 페임 |

## 4. 슈퍼팬 → 일반 팬 유입 플라이휠

```
슈퍼팬이 고품질 그룹 생성
        ↓
탐색 피드 상단 노출 (슈퍼팬 특전)
        ↓
일반 사용자가 발견 → "와, 이거 어떻게 만든 거야?"
        ↓
"나도 만들어보고 싶다" → 가입 + 그룹 생성
        ↓
일부가 과금 시작 → 슈퍼팬으로 성장
        ↓
슈퍼팬이 더 투자 → 더 좋은 콘텐츠 → 반복
```

핵심: 슈퍼팬이 만든 콘텐츠가 플랫폼의 **쇼윈도우** 역할

### 플라이휠 수치화 (목표)

| 단계 | 전환율 목표 |
|------|-----------|
| 탐색 → 가입 | 5% |
| 가입 → 그룹 생성 | 50% |
| 그룹 생성 → 유료 전환 | 15% |
| 유료 → 슈퍼팬 | 10% |

## 5. 슈퍼팬 리텐션 전략

| 전략 | 설명 | 주기 |
|------|------|------|
| 시즌 갱신 | 매 시즌 새 콘셉트 + 한정 아이템 | 4주마다 |
| 랭킹 경쟁 | 시즌별 인기 순위 업데이트 | 주간 |
| 커뮤니티 이벤트 | 크리에이터 간 콜라보/대결 | 월간 |
| 서프라이즈 보상 | 활동량 기반 깜짝 보상 (의상, 콘셉트) | 불규칙 |
| 크리에이터 밋업 | 온라인 밋업 (슈퍼팬 네트워크) | 분기별 |
