# 투자 요청 (Ask)

## 요청 금액

| 구분 | 금액 | 비고 |
|------|------|------|
| 초기창업패키지 | 1억원 | 정부지원금 |
| Pre-seed | 3억원 | (투자 유치 시) |

## 자금 용도

### 1순위: 게임 루프 개발 (50%)
- 시즌 시스템 구현 (드래프트 → 빌드 → 시뮬레이션 → 시즌 결과)
- 관계 구조/케미 시뮬레이션 엔진
- 인증/계정 시스템
- 앱 개발 (iOS/Android)

### 2순위: PMF 검증 (25%)
- Phase 2 UT (시즌 리텐션, 유료 전환 검증)
- 시즌 패스 + 인앱 과금 테스트
- 사용자 세그먼트별 A/B 테스트

### 3순위: 성장 (15%)
- TikTok 마케팅
- 인플루언서 협업
- 해외 시장 테스트 (동남아, 미국)

### 4순위: 운영 (10%)
- AI API 비용
- 인프라
- 기타 운영비

## 실행 방식

듀얼 트랙 애자일 + Lean UX 기반으로 **1주 단위 스프린트**를 반복한다.

```
매주 반복:
  Discovery (발견): 사용자 인터뷰, UT, 데이터 분석 → 다음에 만들 것 결정
  Delivery (구현): 설계 → 개발 → 배포 → 사용자 피드백 수집
  → 매 스프린트마다 Go/Stop 판단
```

## 마일스톤

| 시점 | 목표 | Go/Stop 기준 |
|------|------|-------------|
| Sprint 4-6 (3주) | 시즌 시스템 v1 배포 | 시즌 1 완주율 측정 |
| Sprint 7-12 (6주) | PMF 검증 | "매우 실망" 40%+, 시즌 리텐션 40%+ |
| Sprint 13-18 (6주) | 수익화 테스트 | 유료 전환 15%+, MRR $7,000+ |
| Sprint 19+ | 성장 가속 | MAU 30,000+, 앱 출시 |

매 스프린트마다 사용자 피드백을 반영하여 방향을 조정한다. 고정된 로드맵이 아니라 **검증 결과에 따라 우선순위가 바뀌는 구조**이다.

### Go/Stop 기준 (Sprint 12 시점)

| 판단 | 조건 |
|------|------|
| **Go** | PMF "매우 실망" 40%+, 시즌 리텐션 40%+, 유료 전환 15%+ |
| **Stop** | 위 조건 미달 시 사전 정의된 Pivot 옵션 실행 |

## 왜 지금인가?

1. **기술 타이밍**: AI 생성 비용이 보편화되어 사용자별 고유 캐릭터 생성이 경제적으로 가능
2. **시장 타이밍**: K-pop 드래프트 문화가 글로벌 학습 완료 (프듀, 보이즈 플래닛)
3. **검증 완료**: 2.5년 4번 피봇, 500K 바이럴, Sprint 1 UT에서 카테고리 발견

## 기대 효과

### 투자자에게
- K-pop 팬덤 경제 ($12B+) x AI 시뮬레이션 게임의 교차 시장 진입
- 시즌 축적 = 시간의 함수 → 선점 효과
- 글로벌 확장 가능성 (K-pop 팬 1.5억 명)

### 초기창업패키지에서
- 한국 스타트업의 K-pop x AI 기술 선도
- 글로벌 시장 진출 통한 수출 효과
- AI 엔터테인먼트 신시장 개척
