### 창업 아이템 개요 (요약)
> AI 아이돌을 캐스팅하고 육성하는 시즌 기반 시뮬레이션 플랫폼.
> 팬덤 경제의 3가지 수요(AI 캐릭터 소비, 관계 서사 지출, 개입형 소비)를
> "내가 만들고 키우는" 하나의 경험으로 통합.

### 1. 문제 인식
-> `01-problem.md` 내용 축약
- 팬덤 경제 3가지 수요 (AI 캐릭터, 관계 서사, 개입형 소비)
- 구조적 공백: AI 생성 도구는 1회성, 팬 플랫폼은 소비만, 버추얼 아이돌은 기획사 소유
- UT에서 사용자가 직접 카테고리 정의 ("키우는 게임", "프로듀싱")

### 2. 실현 가능성
-> `02-solution.md` + `03-traction.md`
- Core Game Loop: 드래프트 -> 빌드 -> 시뮬레이션 -> 시즌 결과 -> 반복
- Sprint 1 UT 결과: 그룹 생성 완료율 100%, PD 역할 인지 86%
- 기술 스택: Next.js, FastAPI, GPT, Flux Pro 등
- **사업추진 일정**: Phase 2 마일스톤 표로 변환
- **사업비 집행 계획**: 항목별 금액 표

### 3. 성장전략
-> `04-market.md` + `05-business-model.md`
- TAM: 라이프 시뮬레이션 ~$20B x 버추얼 아이돌 ~$3.9B x AI UGC ~$32.6B 교차
- 수익 모델: 시즌 패스, 드래프트 슬롯, 프리미엄 콘셉트 팩
- Growth Flywheel: 사용자 증가 -> 드래프트 경쟁 -> 선점 가치 -> ARPU 상승
- Moat: K-pop 팬덤 이해 + 시즌 축적 데이터
