# 1. 멤버 이력

## BEFORE (현재)
```
1. 이소연 - CEO - 연세대 HCI/UX 석사(AI-인간 인터랙션 연구), 동덕여대 컴퓨터학과 - 20년 K-pop 팬덤 도메인 전문성, AI 페르소나 관계 형성 사용자 경험 설계, CodeStart 창업(무개발 판매 검증)

2. 김영욱 - CTO - KAIST AI/ML/RL 석사, 서강대 전산학과 - 네이버 nLiveCast P2P 스트리밍 개발(특허 3건 한국/미국/일본), 스캐터랩 PingPong AI 개발(→이루다 2.0, 100만 다운로드), Algorima CEO(연쇄창업, 연 매출 10억+), LiveKit·LangChain 오픈소스 기여자

3. 전제이 - 디자이너 - 경희대 시각디자인과

4. 박제형 - 백엔드 개발자 - 서경대 컴퓨터공학과
```

## AFTER (수정안)
```
1. 이소연 - CEO - 연세대 HCI/UX 석사(AI-인간 인터랙션 연구), 동덕여대 컴퓨터학과 - 20년 K-pop 팬덤 도메인 전문성, AI 페르소나 관계 형성 사용자 경험 설계, CodeStart 창업(무개발 판매 검증)

2. 김영욱 - CTO - KAIST AI/ML/RL 석사, 서강대 전산학과 - 네이버 nLiveCast P2P 스트리밍 개발(특허 3건 한국/미국/일본), 스캐터랩 PingPong AI 개발(→이루다 2.0, 100만DL), Algorima CEO(연쇄창업, 연 매출 10억+), LiveKit·LangChain 오픈소스 기여자

3. [AI 엔지니어 이름] - AI 엔지니어 - [학교/전공] - AI 이미지·영상 콘텐츠 생성 파이프라인 설계·구축, 실시간 음성+립싱크 영상 시스템 개발

4. [백엔드 개발자 이름] - 백엔드 리드 - [학교/전공] - LLM 기반 AI Agent 설계·개발, Text-to-SQL 벤치마크 1위
```

## WHY (변경 이유)
- D2SF는 "학업을 통해 쌓인 기술력"을 중시 -> 이소연의 석사 연구를 "AI-인간 인터랙션 연구"로 구체화
- 김영욱: "기술 개발 총괄" -> "CTO"로 변경 (D2SF는 C-Level 포지션 기준으로 봄)
- 김영욱의 LiveKit/LangChain 오픈소스 기여를 추가 (IR덱에 있는 정보, 기술력 증명)
- 3, 4번: 학교명만 있으면 D2SF 심사역에게 "이 팀의 기술력이 약하다"는 인상을 줌
- IR덱에 "Text-to-SQL 벤치마크 1위", "LLM 기반 AI Agent 설계" 등 강력한 기술 이력이 있는 멤버가 있으므로, 해당 인원으로 교체하거나 경력을 반드시 추가
- **핵심: D2SF는 4명 모두의 "기술적 역량"을 본다. 학교명만 쓰면 감점.**

# 2. 풀고자 하는 문제 (300자)

## BEFORE (현재)
```
K-pop 팬은 서바이벌 투표에 300만원을 쓰지만, 최종 결정은 기획사가 합니다.

2026년 1월, 팬 소통 활동은 13.82% 감소했습니다. 팬들은 과도한 마케팅과 끊임없는 프로모션으로 피로감을 느낍니다. 팬덤은 분열되고 브랜드 충성도는 하락했습니다.

팬은 소비만 하는 게 아니라, 직접 만들고 성장시키는 경험을 원합니다.
```

### 문제점
- "팬 피로감" -> "AI 아이돌 창조"의 논리적 연결이 약함
- D2SF가 중시하는 "기술로 풀 수 있는 문제"라는 프레이밍이 없음
- 시장 크기 체감이 안 됨

## AFTER (수정안)
```
AI 음성·영상 생성 기술이 실시간 수준에 도달하면서, 누구나 고품질 버추얼 캐릭터를 만들 수 있는 시대가 열렸습니다.

K-pop 팬 222개국 3,252만 명이 서바이벌 투표에 참여하지만, 최종 결정권은 기획사에 있고 팬의 기여는 일회성으로 소모됩니다. 5년간 팬 투자 대비 만족도는 지속 하락 중입니다.

Character.AI 사용자 1억 명이 'AI와의 관계 맺기' 수요를 증명했지만, 남의 캐릭터를 소비할 뿐입니다. 내가 직접 AI 아이돌을 창조하고 성장시키는 플랫폼은 아직 없습니다.
```

### WHY
- 첫 문장을 "기술 변화"로 시작 -> D2SF가 보는 "기술로 풀 수 있는 문제" 프레이밍
- 222개국 3,252만 명 -> 시장 크기 체감 (IR덱 데이터 활용)
- Character.AI 1억 명 언급 -> 이미 검증된 시장 + 우리의 차별화
- "아직 없습니다" -> D2SF가 좋아하는 "Frontier/개척자" 키워드

# 3. 핵심 기술 및 제품 경쟁력 (300자)

## BEFORE (현재)
```
실시간 AI 음성+립싱크 영상으로 자연스러운 대화를 구현했습니다. 사용자는 채팅 입력, AI는 음성+표정으로 응답합니다.

팬이 AI 아이돌을 직접 창조합니다. 외모, 성격, 서사까지 설정하고 HCI 연구 기반으로 장기 관계를 형성합니다.

Character.AI는 남의 캐릭터 소비, AIoIA는 내 아이돌 창조+그룹 관리+팬덤 커뮤니티.

TikTok $25로 5,000명, CAC $0.005 검증.
```

### 문제점
- 기술이 "파이프라인 조합"으로 읽힘 (자체 기술이 뭔지 불명확)
- "HCI 연구 기반"의 구체적 내용이 없음
- D2SF 핵심 질문 "이 기술만으로?"에 대한 답이 약함

## AFTER (수정안)
```
핵심 기술은 HCI 연구를 기반으로, AI 캐릭터와 사용자 간 장기적 신뢰와 애착을 형성하는 독자적 인터랙션 시스템입니다. 단순 롤플레이 챗봇이 아닌, 상호작용으로 아이돌의 성격과 서사가 진화하는 성장형 AI 페르소나입니다.

실시간 음성+립싱크 영상 파이프라인으로 AI 아이돌이 표정과 음성으로 자연스럽게 반응합니다.

Character.AI(남의 캐릭터와 1:1 채팅 소비)는 AIoIA(음성·영상, 내 아이돌 창조+팬덤 커뮤니티)로 사용자가 만든 AI 아이돌 데이터가 축적될수록 이탈 불가능한 락인을 만듭니다.
```

### WHY
- **첫 문장을 "자체 개발한 AI 페르소나 관계 형성 엔진"으로 시작** -> D2SF가 원하는 "이 팀만의 고유 기술" 즉시 전달
- "독자적 인터랙션 시스템" -> 기술적 해자 강조
- "성장형 AI 페르소나" -> 기존 챗봇과의 기술적 차별점 명확화
- "데이터 락인" -> 기술적 모트 (D2SF가 투자한 반달AI의 OmniPub 포맷처럼, 데이터 구조 자체가 모트)
- OpenAI, API 등 외부 의존성 언급 완전 삭제

# 4. 사업모델 및 계획 (300자)

## BEFORE (현재)
```
월 3.99달러 구독으로 무제한 AI 아이돌 생성과 음성 대화를 제공합니다.

Phase 1 (현재): 개인 사용자 구독 (Character.AI 9.99달러, 버블 4,500원 참고)
Phase 2 (6개월): 크리에이터 수익 공유 - YouTube 70/30 모델처럼 팬이 만든 AI 아이돌 콘텐츠로 수익 창출
Phase 3 (12개월): B2B - 소속사/MCN에게 Enterprise API 제공 (월 1만~5만 달러)

초기 시장: 미국 K-pop 팬 35만명 × 월 24달러 = 연 1억 달러 시장
```

### 문제점
- 나쁘지 않지만, D2SF가 중시하는 "글로벌" 키워드가 약함
- Phase 구조는 유지하되, 글로벌 퍼스트를 더 강조

## AFTER (수정안)
```
- Phase 1 (현재): 월 $3.99 구독 - 무제한 AI 아이돌 생성, 음성·영상 대화. 실 사용자 500명, TikTok CAC $0.005 검증 완료
- Phase 2 (6개월): 크리에이터 수익 공유 - AI 아이돌 콘텐츠로 수익 창출 (YouTube 70/30 모델)
- Phase 3 (12개월): B2B 엔터사/MCN에게 AI 아이돌 생성 Enterprise API 제공 

- 초기 타겟: 미국 K-pop Gen Z 팬 35만명, SOM $100M, AI 영상 스트리밍 기술 상용화에 맞춰 12개월 내 라이브 스트리밍으로 확장
```

### WHY
- 초기 타겟을 미국 K-pop Gen Z 팬으로 명시 -> D2SF 포트폴리오 80%가 글로벌 타겟이라는 점에 직접 호응
- 트랙션 데이터(실 사용자 500명) 추가 -> IR덱에서 검증된 수치
- SOM $100M 명시 -> 현실적인 시작점과 향후 SAM/TAM 확장 가능성 보여주기
- "AI 영상 스트리밍 기술 상용화" -> D2SF가 좋아하는 "기술 트렌드에 올라타기"

# 5. 기대하는 네이버와의 협업 (300자)

## BEFORE (현재)
```
1. HyperCLOVA X로 한국어 대화 품질 최적화 - 현재 OpenAI API 사용 중, 네이버 AI 인프라로 전환 희망

2. CLOVA Speech로 AI 아이돌 목소리 생성 - 실시간 음성 대화에 필수 기술

3. 포지셔닝 차별화 - 네이버 웹툰 캐릭터챗(웹툰 IP, 350만 사용자)과 AIoIA(K-pop IP, UGC 플랫폼)는 타겟이 다름

4. 글로벌 확장 지원 - 네이버 클라우드 인프라로 미국/동남아 서비스, D2SF 네트워크로 후속 투자 연결
```

### 문제점 (가장 큰 수정 필요)
- **"현재 OpenAI API 사용 중"** -> D2SF에게 "자체 기술 없다"는 자백. 즉시 삭제.
- **캐릭터챗 언급** -> "우리 서비스와 겹치네?"라는 인식을 심어줌. 즉시 삭제.
- 전체적으로 "네이버에게서 받을 것"만 나열 -> "우리가 네이버에게 줄 수 있는 것"이 없음
- D2SF 양상환 센터장 인터뷰: 투자 기준은 "협업", "글로벌 진출", "신시장 개척" -> 이 3가지에 맞춰야 함

## AFTER (수정안)
```
1. K-pop AI UGC 신시장: AI 아이돌 생성 콘텐츠는 네이버 플랫폼에서 아직 없는 새로운 UGC 카테고리입니다. 글로벌 K-pop Gen Z의 AI 콘텐츠 소비 데이터와 팬덤 행동 인사이트를 공유합니다.

2. HyperCLOVA X·CLOVA Speech 활용: 한국어 AI 아이돌 대화 품질과 음성 자연스러움을 극대화하고, 실사용자의 대화 피드백으로 네이버 AI 모델에 기여합니다.

3. 네이버 클라우드 인프라로 미국·동남아 글로벌 서비스 스케일링, D2SF 네트워크로 북미 시장 확장을 가속합니다.
```

### WHY
- **"OpenAI API 사용 중" 완전 삭제** -> 자체 기술이 있다는 인상 유지
- **캐릭터챗 비교 완전 삭제** -> 내부 서비스 충돌 우려 원천 제거
- **"네이버에 줄 수 있는 것"을 먼저 배치:**
  - "새로운 UGC 카테고리" -> 네이버에 없는 것을 우리가 만든다 (양방향 가치)
  - "글로벌 K-pop Gen Z 데이터·인사이트 공유" -> 네이버 글로벌 전략에 구체적 기여
  - "실사용자 피드백으로 네이버 AI 모델 발전 기여" -> 단방향 수혜가 아닌 상호 가치
- **D2SF 실리콘밸리 네트워크 언급** -> D2SF가 2024년 10월 개설한 실리콘밸리 사무소를 활용하겠다는 구체적 연결

# 7. 추가 권고사항

## 7.1 기술/제품 분류 선택
현재 선택한 분류가 뭔지 모르겠지만, 추천:
- 1순위: **AI/데이터** (D2SF 투자의 54%가 AI)
- 2순위: **콘텐츠** (K-pop 콘텐츠 플랫폼)

## 7.2 IR 덱에서 반드시 보강할 페이지
- **Traction 페이지**: 1개월 리텐션 5%는 약한 수치. 가능하다면 "코어 사용자(주 3회 이상 접속)의 리텐션"이나 "NPS 점수" 등 더 나은 지표를 사용하거나, 리텐션 개선 추이를 보여주기
- **기술 아키텍처 페이지**: IR 덱에 기술 아키텍처 다이어그램이 없음. D2SF는 2차 미팅에서 네이버 기술조직이 리뷰 -> 기술 스택과 자체 개발 범위를 보여주는 페이지 필수

## 7.3 투자 유치 히스토리 기재
지원서에 별도로 기재해야 하는 항목:
- 형식: "Pre-Seed - 26년 - X억원 - Y억원 - Z벤처스 / N억원 투자유치 희망, M억원 밸류"
- 팁: D2SF는 시드 투자 전문이므로, 희망 밸류를 너무 높게 잡지 않는 것이 좋음
