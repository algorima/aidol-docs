# 시장 분석 (Market)

## TAM/SAM/SOM

### TAM (Total Addressable Market)

AIdol은 세 시장의 교차점에 위치한다.

| 시장 | 규모 | 근거 |
|------|------|------|
| 라이프 시뮬레이션 게임 | **~$20B** | The Sims 프랜차이즈 누적 $5B+, 모바일 시뮬레이션 성장 |
| 버추얼 아이돌/버튜버 | **~$3.9B** | PLAVE $32.5M, 버튜버 시장 전체 |
| AI UGC (AI 생성 콘텐츠) | **~$32.6B** | Character.AI 2천만 MAU, AI 캐릭터 생성 시장 |

**교차 TAM**: 이 세 시장이 겹치는 "AI 캐릭터 시뮬레이션" 영역

## 시장 트렌드

### K-pop 산업
- 글로벌 K-pop 시장 규모: $12B+ (2025)
- 연평균 성장률: 15%+
- 해외 팬 비중 증가 (특히 동남아 62%, 미국, 유럽)
- 서바이벌/드래프트 문화가 글로벌 표준으로 정착

### 버추얼 아이돌 검증
- PLAVE: 가상 아이돌 그룹, 연매출 **$32.5M**
- 나비스: AI 기반 버추얼 아이돌
- 팬들이 버추얼 존재에 실제 아이돌과 동일한 수준으로 투자한다는 것을 검증

### AI 캐릭터 시장 성장
- Character.AI: 2천만 MAU, 평균 세션 2시간
- AI 채팅앱 다운로드 88% YoY 증가
- AI 생성 비용 하락 → 유저별 고유 캐릭터 생성이 경제적으로 가능

### 개입형 콘텐츠 수요
- 보이즈 플래닛: 3,252만 표
- 팬 참여형 콘텐츠의 engagement가 일방적 콘텐츠 대비 압도적

## 경쟁 환경

| 서비스 | 카테고리 | 강점 | 약점 | AIdol 차별점 |
|--------|----------|------|------|-------------|
| **alora** | AI 아이돌 채팅 | K-pop 특화, 공식 IP | 소비만 가능, 생성/육성 없음 | 유저가 직접 캐스팅+육성 |
| **제타 (ZETA)** | 버추얼 아이돌 팬 플랫폼 | 기존 팬덤 연결 | 기획사 소유 IP, 유저 개입 없음 | 유저 소유 IP |
| **mini ai** | AI 캐릭터 생성 | 쉬운 생성 UX | 1회성 생성, 시즌/성장 없음 | 시즌 기반 축적 |
| **크랙 (Krak)** | AI 셀럽 챗봇 | 실제 셀럽 IP | 라이선스 의존, 유저 IP 아님 | 자체 IP, 유저 소유 |
| **월간 아이돌** | 아이돌 육성 게임 | 육성 메카닉 | AI 아닌 정적 콘텐츠 | AI 기반 동적 시뮬레이션 |

### 포지셔닝

```
           소비 전용          생성 + 육성
           ─────────────────────────────
기획사 IP │  Bubble, ZETA     │            │
          │  alora, 크랙      │            │
          ├──────────────────┤            │
유저 IP   │  Character.AI     │  AIdol ★   │
          │  mini ai          │            │
           ─────────────────────────────
```

AIdol은 **"유저 IP + 생성 + 육성"** 교차점의 유일한 플레이어이다.

## Moat (경쟁 우위)

### 1. K-pop 팬덤 이해
- 1.5년간 4번의 피봇을 통해 축적한 K-pop 팬 행동 데이터
- 팬덤 용어, 관계 구조, 콘셉트 문법에 대한 도메인 지식

### 2. 시즌 축적 데이터 (시간의 함수)
- 유저의 그룹 히스토리, 조합 패턴, 관계 로그가 쌓일수록 전환비용 상승
- 경쟁사가 Day 1에 복제할 수 없는 자산
- **"내가 3시즌 동안 키운 그룹"을 버리고 다른 서비스로 갈 수 없다**
