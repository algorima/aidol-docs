# 솔루션 (Solution)

## 핵심 가치 제안

> **"AI 아이돌을 캐스팅하고 키우는 시뮬레이션 게임"**

유저가 프로듀서가 되어 AI 아이돌 그룹을 드래프트하고, 빌드하고, 시뮬레이션하고, 시즌 결과를 확인하는 반복 루프. 소비가 아니라 **설계와 육성**이 핵심 경험이다.

### 제품명
- **AIdol**: AI 아이돌 시뮬레이션 플랫폼

## 1. Core Game Loop

```
드래프트 (Draft)
  → 연습생 풀에서 AI 아이돌을 캐스팅. 등급, 능력치, MBTI, 포지션을 보고 전략적으로 선택
  → 경쟁 캐스팅: 한 번 선택된 연습생은 다른 유저가 선택 불가

빌드 (Build)
  → 그룹 콘셉트 설정, 포지션 배정, 관계 구조 설계
  → 멤버 조합에 따라 케미/시너지가 달라짐

시뮬레이션 (Simulate)
  → AI 아이돌과 1:1 대화, 그룹 내 관계 진행
  → 멤버 성장, 이벤트 발생, 팬 반응 시뮬레이션

시즌 결과 (Season Result)
  → 시즌 종료 시 성과 평가 (인기, 실력 성장, 팬 수)
  → 랭킹, 히스토리 기록

반복 (Next Season)
  → 새 시즌에 신규 연습생 드래프트, 멤버 교체/영입
  → 이전 시즌 데이터 축적 → 그룹 IP 진화
```

## 2. MVP 테스트 결과 (Sprint 1, 인터뷰 7명)

### 2.1 긍정 신호

| 지표 | 결과 |
|------|------|
| 그룹 생성 완료율 | **100%** (7/7) |
| 캐스팅 고민 점수 | **평균 4.2/5** |
| PD 역할 인지 | **6/7 (86%)** |
| B그룹(크리에이터) 몰입도 | **4-5/5** |

- AI 아이돌 제작에 몰입: 등급 전략 고민, 포지션 밸런스 설계, MBTI 비율 조정
- 비팬(U3)도 프로듀싱 경험으로 강하게 몰입
- "키운다" 맥락 추가 시 공유 의향 급상승

### 2.2 위험 신호

| 위험 | 근거 |
|------|------|
| 단순 AI 챗봇 인식 | 채팅만 제공하면 Character.AI와 구분 불가 |
| 관계 구조 얕으면 이탈 | U7: "퍼즐 끼워 맞춘 느낌" → 애착 형성 실패 |
| 1회성 체험 종료 | 성장/시즌 맥락 없으면 재방문 동인 부재 |

### 2.3 결론

> **"실제 아이돌을 대체하지 않는다. 팬이 실험하고 확장할 수 있는 팬덤 시뮬레이션 공간이다."**

핵심 차별점은 "AI 친구"가 아니라 **"내가 캐스팅하고 키우는 시뮬레이션"**이라는 카테고리. 유저가 이 카테고리를 직접 정의했다.

## 3. Meta Loop: 시즌 축적 → 플랫폼화

### 3.1 시즌 축적

| 축적 데이터 | 가치 |
|------------|------|
| 조합 히스토리 | 어떤 멤버 조합이 성공하는지 학습 |
| 관계 로그 | 유저-아이돌 간 대화/이벤트 히스토리 |
| 성장 데이터 | 시즌별 성과, 전략 패턴 |
| 유저 소유 IP | 유저가 만든 그룹 = 유저의 자산 |

### 3.2 플랫폼 효과

```
시즌 축적 → 전환비용 상승 (내 그룹 히스토리를 버릴 수 없다)
         → 추천 개인화 (유저 취향 학습 → 더 정확한 연습생 추천)
         → 네트워크 효과 (유저 그룹 간 경쟁/비교 → 커뮤니티 형성)
```

## 4. 현재 MVP 기능

### 4.1 아이돌 생성

#### 그룹 기획
- 그룹명 (필수), 콘셉트 (선택)
- 엠블럼 이미지 AI 생성 (512x512 PNG)

#### 프리셋 연습생 캐스팅
- 시스템 제공 연습생 풀
- 성별 필터 (남성/여성/혼성)
- 등급 (A/B/C/F), 능력치 (육각형 차트)
- 경쟁 캐스팅: 선착순, 한 번 캐스팅되면 다른 유저 선택 불가

#### 신규 멤버 생성
- 성별 선택, MBTI 4개 슬라이더
- 이름 입력, 프로필 이미지 AI 생성

#### 포지션 배정
- 리더, 메인보컬, 서브보컬, 메인댄서, 서브댄서, 메인래퍼, 서브래퍼, 비주얼, 막내

### 4.2 채팅
- 유저 <> 멤버 1:1 대화
- 생성 시 입력한 성격 기반 페르소나 응답
- 대화 히스토리 유지

### 4.3 관리/공유
- 내 그룹 목록 관리
- URL 공유 (조회만, 대화 불가)
- "나도 만들기" CTA

## 6. Why Now

### 6.1 AI 생성 비용 commodity화
- 이미지 생성, 음성 합성, 텍스트 생성 비용이 급격히 하락
- 2년 전에는 불가능했던 "유저별 고유 AI 캐릭터 생성"이 경제적으로 가능해짐

### 6.2 K-POP 드래프트 문화 글로벌 학습 완료
- 프로듀스 101 → 보이즈 플래닛 → 글로벌 팬들이 "투표하고 뽑는" 문화를 이미 체득
- "국민 프로듀서"라는 역할에 대한 사전 학습이 끝남

### 6.3 수요 측 IP 플랫폼 부재
- 공급자(기획사) 주도 IP만 존재
- 유저가 만들고 소유하는 IP 플랫폼은 아직 없음
- PLAVE($32.5M 매출)가 버추얼 아이돌 수요를 검증했으나, 여전히 기획사 소유 모델
