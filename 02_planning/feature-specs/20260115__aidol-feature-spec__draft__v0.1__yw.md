## Sprint 1 화면

1. **랜딩 페이지**: 서비스 첫 진입 화면. 간판 아이돌 영상과 CTA 버튼 제공
2. **연습생 선발 페이지**: 프리셋 연습생 목록 조회 및 신규 멤버 추가 진입점
3. **신규 멤버 추가 페이지**
4. **캐스팅 완료 페이지**: 캐스팅한 연습생 프로필, 남은 캐스팅 슬롯 수
5. **캐스팅 보드**: 지금까지 캐스팅한 모든 연습생 목록 표시
6. **포지션 배정**: 데뷔조 확정 멤버들에게 포지션 배정
7. **그룹 기획**
8. **그룹 기획 완료**
9. **UI/UX 공통 사항**

## 논의되었으나 Sprint 1 제외 확정 항목

1. **관계성 설정 / 멤버 간 대화**: 이번 MVP 범위 초과
2. **연습생과 직접 대화**: 기술적 난이도, 스프린트 2에서는 한번 고민
3. **목소리 생성**: 콘텐츠 양으로 대체
4. **월말 평가 시스템**: 향후 확장
5. **계약금/예산 시스템**: 현재 개념 없음
6. **실시간 조회 중 표시**: 구현 난이도 까다로움
