## 개요

Sprint 1은 서비스의 핵심 플로우인 **AI 아이돌 그룹 생성**을 구현합니다.  
사용자가 연습생을 캐스팅하고, 신규 멤버를 생성하고, 포지션을 배정하여 그룹을 완성하는 전체 여정을 포함합니다.

---

서비스 첫 진입 화면으로, 간판 아이돌 영상과 CTA 버튼을 제공합니다.

### 1-1. 서비스 소개 타이틀

| 항목 | 내용 |
|------|------|
| **설명** | 서비스 첫 진입 화면, 간판 아이돌 영상과 CTA 버튼 제공 |
| **기능 명세** | - 메인 타이틀: "1+∞ 무한한 가능성의 아이돌이 모이는 곳"<br>- 완벽한 아이돌을 꿈꾸는 AI 연습생들과 함께 무한하게 진화하는 당신의 플래닛<br>- 글로벌 팬덤이 직접 캐스팅하고 데뷔시키는 AI 아이돌 그룹 넥스트 제너레이션 프로젝트<br>- 스타 크리에이터님!<br>- 당신이 만들어갈 AI 아이돌에게 투표하세요!<br>- 간판 아이돌 영상 (자동 재생, 루프)<br>- "연습생 캐스팅하기" CTA 버튼 |

## 2. 연습생 카드 그리드 (연습생 선발 페이지)

### 2-1. 성별 필터

| 항목 | 내용 |
|------|------|
| **설명** | - 연습생 목록 화면 하단<br>- 남성 (기본값) / 여성 / 혼성 |
| **기능 명세** | 필터 선택 시 해당 성별 연습생만 목록에 표시 |

### 2-2. 프리셋 연습생 카드 목록

| 항목 | 내용 |
|------|------|
| **설명** | 프로필 사진, 이름, 등급 (A/B/C/F) |
| **기능 명세** | - 콘텐츠 수 한정 ?개 + 중간/맨 아래 신규 멤버 추가<br>- 등급 높은 순 → 최근 추가 순 |

### 2-3. 연습생 상태 표시

| 항목 | 내용 |
|------|------|
| **설명** | - **선택 가능**: 일반 카드<br>- **계약 완료**: 블러 처리 + "계약 완료" 레이블 |

### 2-4. 연습생 등급 기준

| 항목 | 내용 |
|------|------|
| **설명** | 연습생 능력치 평균 점수 |
| **기능 명세** | - A(80~100점): 최상위 실력, 데뷔 준비 완료 수준<br>- B(60~79점): 우수한 실력, 추가 트레이닝 필요<br>- C(40~59점): 보통 실력, 성장 잠재력 있음<br>- F(0~39점): 신입/초보, 집중 트레이닝 필요<br>※ 신규 생성 연습생: 등급 F, 모든 능력치 0에서 시작 |

### 2-5. 연습생 상세 프로필 팝업

| 항목 | 내용 |
|------|------|
| **설명** | 카드 클릭 시 상세 프로필 표시 → 체류 시간 증가 목적 |
| **기능 명세** | - 트리거: 연습생 카드 클릭<br>- 표시 정보: 프로필 사진, 이름, 등급, MBTI, 능력치 (6각형 차트)<br>- 간단한 자기 소개 및 서사<br>- 능력치 항목: 보컬 / 댄스 / 랩 / 비주얼 / 체력 / 매력<br>- CTA 버튼: "캐스팅 후보로 추가" |

## 3. 신규 멤버 추가 (신규 멤버 추가 페이지)

### 3-1. 성별 선택 (Step 1)

| 항목 | 내용 |
|------|------|
| **설명** | 신규 멤버 추가 버튼 누르면 전환 |
| **기능 명세** | 남성 / 여성 |

### 3-2. 성격 설정 - MBTI 슬라이더 (Step 2)

| 항목 | 내용 |
|------|------|
| **설명** | - 성별 선택을 완료하고 '다음' 버튼 누르면 전환<br>- 슬라이더 4개 (각 MBTI 축) |
| **기능 명세** | - 에너지 방향: I (내향) ↔ E (외향)<br>- 인식 기능: S (감각) ↔ N (직관)<br>- 판단 기능: T (사고) ↔ F (감정)<br>- 생활 양식: J (판단) ↔ P (인식) |

### 3-3. 성격 설정 - MBTI 슬라이더 지표

| 항목 | 내용 |
|------|------|
| **설명** | **중립 선택 불가 처리** |
| **기능 명세** | 슬라이더 단계: 짝수 (1부터 10까지, 10단계) |

### 3-4. 이름 입력 (Step 3)

| 항목 | 내용 |
|------|------|
| **설명** | MBTI 설정을 완료하고 '다음' 버튼을 누르면 전환 |
| **기능 명세** | - 텍스트 인풋 (최대 10자)<br>- 빈 값 불가, 특수문자 제한 |

### 3-5. 프로필 사진 생성 (Step 4)

| 항목 | 내용 |
|------|------|
| **설명** | 프롬프트 텍스트 입력 |
| **기능 명세** | - "이미지 생성" 버튼 클릭 → 로딩 → 생성된 이미지 하단 표시<br>- 마음에 들지 않으면 프롬프트 수정 후 재생성 가능<br>- "캐스팅 완료" 버튼으로 다음 단계 이동 |

## 논의되었으나 Sprint 1 제외 확정 항목

1. **관계성 설정 / 멤버 간 대화**: 이번 MVP 범위 초과
2. **연습생과 직접 대화**: 기술적 난이도, 스프린트 2에서는 한번 고민
3. **목소리 생성**: 콘텐츠 양으로 대체
4. **월말 평가 시스템**: 향후 확장
5. **계약금/예산 시스템**: 현재 개념 없음
6. **실시간 조회 중 표시**: 구현 난이도 까다로움
