### 9.1 PR 리뷰

- 리뷰 요청 후 **48시간 이내** 리뷰 완료
- 48시간 초과 시 리뷰어를 변경하거나 소연이 직접 머지 판단

### 9.2 주간 체크인

- 매주 수요일 Sprint 완료 시점에 진행 상황 공유
- 차단 요인(blocker) 있으면 즉시 Slack 공유, 수요일까지 기다리지 않기

### 9.3 문서 작업

- 기능 명세 변경 시 PR로 올리고 관련 담당자 태그
- ERD/API 변경은 반드시 영욱 리뷰 필수
