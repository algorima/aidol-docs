## 1. 현재 상태 (2026년 2월 기준)

### 1.1 제품 개발 진행 상황

| 단계 | 기간 | 상태 |
|------|------|------|
| Discovery Sprint 1 | ~1/23 | 완료 |
| Delivery Sprint 1 + Discovery Sprint 2 | 1/24~1/30 | 완료 |
| Delivery Sprint 2 + Discovery Sprint 3 | 1/31~2/6 | 완료 |
| Delivery Sprint 3 | 2/7~2/13 | 진행중 |
| MVP 출시 목표 | 2/16 (설 전) | - |

### 1.2 이미 검증된 데이터

**바이럴 가능성**
- TikTok 단일 영상 500K 조회 (5일)
- 5,000+ 리드 수집
- 500+ 가입자
- 마케팅 비용: $0

왜 중요한가: 유료 광고 없이 5일 만에 500명이 가입했다는 건 제품에 대한 자연 수요가 있다는 증거다. CAC(고객획득비용)가 0에 가깝다는 뜻이기도 하다.

**유료 전환 가능성**
- 워크샵 참가자 20명 중 6명 결제 (30% 전환율)
- 객단가: 150,000원/인
- 누적 매출: ~$500

왜 중요한가: 초기 사용자 중 30%가 150,000원을 지불했다는 건 가격 민감도가 낮다는 의미다. 일반적인 freemium 전환율 2-5%를 크게 상회한다.

**감정적 연결 가능성**
- AI Boyfriend 실험 (Pivot 1): 300명 중 75%가 AI에게 진심 어린 애정 표현
- BTS AI 실험 (Pivot 2): K-pop 팬들이 15분 만에 깊은 이야기 공유

왜 중요한가: AI와의 대화가 단순 정보 교환이 아닌 감정적 교류로 이어진다는 걸 확인했다. 이건 리텐션의 핵심 동력이다.

**자발적 홍보 가능성**
- 마케팅 $0 실험: 대학생 19명이 무료로 홍보 콘텐츠 제작
- 결과: 400명+ 방문자 유도

왜 중요한가: 사용자가 제품을 알아서 홍보한다는 건 바이럴 계수가 1 이상일 가능성이 있다는 뜻이다.

## 2. 목표 지표와 근거

### 2.1 D7 Retention 15%

**업계 벤치마크**
- Adjust.com 데이터: 모바일 앱 평균 D7 Retention 13%
- Social/Entertainment 앱: D1 29%, D7 15% 정도가 상위권
- Character.AI: 평균 세션 시간 2시간 (높은 engagement의 증거)

**우리의 근거**
- AI Boyfriend 실험에서 75%가 진심 어린 대화 → 높은 몰입도 가능성
- K-pop 팬덤의 특성: 일반 사용자보다 충성도가 높음
- 슈퍼팬 연간 지출 $1,422 (일반 소비자 대비 압도적)

**15%를 선택한 이유**
- 업계 평균(13%)을 살짝 상회하는 수준
- 너무 높으면(20%+) 달성 불가능, 너무 낮으면(10%) PMF 신호로 부족
- Character.AI도 초기에는 비슷한 수준이었다가 제품 개선으로 상승

### 2.2 "매우 실망" 응답 40%

**근거: Sean Ellis / Superhuman PMF 측정법**
- 질문: "이 제품을 더 이상 사용할 수 없다면 어떤 기분이겠습니까?"
- 응답 중 "매우 실망"이 40% 이상이면 PMF 달성으로 판단
- Superhuman은 이 방법으로 PMF를 찾아 유니콘이 됨

**40%를 선택한 이유**
- Sean Ellis가 수백 개 스타트업을 분석해서 도출한 경험적 기준
- 40% 미만: 제품 개선 필요
- 40% 이상: 성장에 집중해도 됨

### 2.3 가입자 1,000명

**근거**
- 5일에 500명 (마케팅 $0)
- 단순 계산: 30일이면 3,000명 가능
- 보수적으로 1,000명 목표 (TikTok 알고리즘 변동성 고려)

**1,000명을 선택한 이유**
- 통계적 유의성: 설문 조사 시 신뢰구간 95%, 오차범위 3%를 위해 필요한 표본 크기
- PMF 설문에서 의미 있는 결과를 얻으려면 최소 1,000명 필요

### 2.4 유료 전환 의향 10%

**근거**
- 워크샵에서 30% 전환율 기록
- 하지만 워크샵 참가자는 이미 관심이 높은 사람들 (선택 편향)
- 일반 사용자는 더 낮을 것으로 예상

**10%를 선택한 이유**
- 워크샵 30%의 1/3 수준 (보수적 추정)
- 일반적인 freemium 전환율 2-5%보다는 높게 설정
- K-pop 팬덤의 높은 지불 의향 고려

## 3. Phase 1: 핵심 경험 검증 (Sprint 1-3)

### 3.1 가설

"나만의 아이돌을 만들고 대화하는 경험이 사용자를 만족시킨다"

### 3.2 실제 구현 범위

**Sprint 1: 아이돌 생성 (1/24~1/30, 완료)**
- 랜딩 페이지 (HeroSection)
- 캐스팅 페이지: 성별 필터, 연습생 카드 그리드, 상세 프로필 팝업
- 신규 멤버 생성: 5단계 플로우 (성별→MBTI→이름→외모→이미지 생성)
- 캐스팅 보드: 선발된 멤버 목록, 삭제 기능
- 포지션 배정: 9개 포지션 (리더, 메인보컬 등)

**Sprint 2: 관계 설정 (1/31~2/6, 완료)**
- 관계 지도 프리셋: 포지션/나이 기반 자동 관계 생성
- 캐릭터 서사 설정: 자유 텍스트 입력 → AI가 프롬프트화
- 관계 디테일 설정: 사용자 커스텀 관계 추가
- 관계 보드: 노드-엣지 시각화

**Sprint 3: 채팅 (2/7~2/13, 진행중)**
- 내 그룹 페이지: 채팅 진입점
- 수신함 페이지: 메시지 목록
- 채팅 화면: 1:1 대화, 메시지 발신/수신

### 3.3 검증 항목과 측정 방법

| 항목 | 목표 | 측정 방법 | 왜 이 지표인가 |
|------|------|----------|---------------|
| 생성 완료율 | 70%+ | 온보딩 퍼널 (PostHog) | 첫 경험의 마찰 측정 |
| 첫 대화 시작률 | 80%+ | 이벤트 추적 | 핵심 기능 도달률 |
| D7 Retention | 15%+ | 코호트 분석 | 재방문 의향 |
| 가입자 수 | 1,000명+ | GA4 | 시장 수요 확인 |

### 3.4 리스크와 대응

| 리스크 | 확률 | 영향 | 대응 |
|--------|------|------|------|
| 이미지 생성 품질 낮음 | 중 | 사용자 이탈 | 프리셋 스타일 제공, 재생성 기능 |
| TikTok 알고리즘 변동 | 중 | 가입자 목표 미달 | 다른 채널(Instagram, Twitter) 병행 |
| 채팅 응답 시간 지연 | 저 | UX 저하 | 스트리밍 응답 구현 (Phase 2) |

### 3.5 GO/STOP 기준

**GO (2월 말 판단)**
- 가입자 >= 1,000명
- D7 Retention >= 15%
- 생성 완료율 >= 70%

**STOP**
- 가입자 < 300명
- D7 Retention < 7%
- 생성 완료율 < 30%

## 5. Phase 3-12: 성장 및 수익화

### 5.1 Phase 3 (Sprint 7-10): 유료 전환 검증

**가설**: 사용자가 프리미엄 기능에 돈을 지불할 의향이 있다

**구현 범위**:
- 프리미엄 기능 정의 (추가 이미지 생성, 음성 메시지 등)
- 가격 테스트 (Van Westendorp 방법론)
- 결제 시스템 연동 (Stripe)

**목표**: 유료 전환 의향 10%+, 최적 가격대 도출

### 5.2 Phase 4 (Sprint 11-13): 구독 모델

**가설**: 구독 모델로 반복 매출이 가능하다

**구현 범위**:
- 월 구독 플랜 ($5, $10, $20 티어)
- 구독 관리 시스템
- 이탈 방지 시스템 (cancel flow 최적화)

**목표**: MRR $500+, 구독 이탈률 10% 미만

### 5.3 Phase 5 (Sprint 14-16): 육성 시스템

**가설**: 육성 시스템이 장기 리텐션을 높인다

**구현 범위**:
- 친밀도 시스템 (대화할수록 친해짐)
- 레벨업 시스템 (멤버 능력치 성장)
- 일일 퀘스트/미션

**목표**: D30 Retention 10%+

### 5.4 Phase 6 (Sprint 17-19): 음성 기능

**가설**: 음성/영상 통화가 유료 전환율을 높인다

**구현 범위**:
- TTS (Text-to-Speech) 음성 메시지
- 음성 통화 (1:1)
- 음성 커스터마이징

**목표**: 음성 사용률 30%+

### 5.5 Phase 7 (Sprint 20-23): 커뮤니티

**가설**: 커뮤니티/공유 기능이 바이럴 성장을 가속한다

**구현 범위**:
- 그룹 공유 기능
- 팬아트 갤러리
- 다른 사용자 그룹 팔로우

**목표**: 바이럴 계수 1.2+

### 5.6 Phase 8-12: 확장

- Phase 8: 해외 확장 (동남아, 미국)
- Phase 9: 모바일 앱 출시
- Phase 10: B2B 모델 (기획사 협업)
- Phase 11: Unit Economics 최적화
- Phase 12: Series A 준비

### 6.1 타이밍

- 시드 투자 시기: 2026년 Q3 (7-9월)
