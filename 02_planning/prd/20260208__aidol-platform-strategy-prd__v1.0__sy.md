## 1. Executive Summary

### 1.1 Vision

**"AI 시대의 유튜브/틱톡 - AI 아이돌 콘텐츠 UGC 플랫폼"**

AIoIA는 사용자가 AI 아이돌을 생성하고, 자동 생성된 콘텐츠를 공유하며, 바이럴을 통해 성장하는 크리에이터 이코노미 플랫폼이다.

### 1.2 우리는 무엇이 아닌가

- ❌ K-pop 특화 Character.AI
- ❌ 1:1 AI 컴패니언 서비스
- ❌ B2B SaaS 도구 (소속사 대상)

### 1.3 전략적 접근

**단계적 검증 및 확장:**

```
Phase 1 (0-6개월): 개인 AI 컴패니언 검증
  ↓
Phase 2 (6-12개월): UGC 콘텐츠 플랫폼 전환
  ↓
Phase 3 (12개월+): B2B 확장
```

**Why Phase 1이 필요한가?**
1. 콘텐츠 자동 생성 기술 미완성 (개발 필요)
2. Chicken-and-egg 초기 공급 문제 (프리셋으로 해결)
3. 핵심 가설 검증: "AI 아이돌 만드는 게 재미있는가?" + "다른 사람 콘텐츠 보고 싶은가?"

## 2. Problem & Solution

### 2.1 시장 기회

**기존 서비스의 한계:**

| 서비스 | 문제점 | AIoIA 해결 |
|--------|--------|-----------|
| Character.AI | 1:1 채팅만, 바이럴 불가 | 영상 콘텐츠 + 공유 |
| YouTube/TikTok | 콘텐츠 제작 진입장벽 높음 | AI 자동 생성 |
| 버추얼 유튜버 (VTuber) | 사람이 직접 운영 | AI가 자동 생성 |

**핵심 차별점:**
- **낮은 진입장벽**: AI가 콘텐츠 자동 생성 → 누구나 크리에이터
- **바이럴 가능성**: 영상 콘텐츠 + 공유 → 유튜브/틱톡 모델
- **확장성**: 유저가 공급 → 무한 콘텐츠

### 2.2 Chicken-and-Egg 해결

**문제:**
```
콘텐츠 없음 → 소비자 안 옴
소비자 없음 → 크리에이터 안 만듦
```

**우리의 해법:**

**Phase 1: Seed + Validate**
```
1. Alora가 프리셋 AI 아이돌 제공 (초기 공급)
   ↓
2. 유저가 "다른 그룹" 콘텐츠 소비
   ↓
3. "나도 만들고 싶다" (생성)
   ↓
4. 내 AI 아이돌과 1:1 채팅
   ↓
5. 공유 의향 측정 (UT 설문)
```

**검증 지표:**
- 소비 지표 높음 → 수요 증명
- 전환율 높음 (30%+) → Chicken-and-egg 해결 가능
- 공유 의향 높음 (4.0+) → 공급 의지 증명

**Phase 2: Self-Sustaining Loop**
```
유저 A가 AI 아이돌 생성
  ↓
콘텐츠 자동 생성 + 업로드
  ↓
유저 B가 소비
  ↓
"나도 만들고 싶다"
  ↓
유저 B가 크리에이터로 전환
  ↓
더 많은 콘텐츠 → 플랫폼 성장
```

## 3. Phase별 전략

### 3.1 Phase 1: 개인 AI 컴패니언 (현재 - 6개월)

#### 3.1.1 목표

**"생성의 재미" + "소비 욕구" + "공유 의향" 검증**

#### 3.1.2 비즈니스 모델

```
유저가 AI 아이돌 그룹 생성 (캐스팅)
  ↓
프리셋 or 커스텀 AI 아이돌
  ↓
1:1 채팅 + 영상 콘텐츠 소비
  ↓
프리미엄 콘텐츠 구독 ($3.99/월)
```

**수익 구조:**
- 구독료: $3.99/월 (프리셋 생성 + 1:1 채팅)
- 프리미엄: 독점 콘텐츠, 굿즈 기획 권한

#### 3.1.3 핵심 기능

| 카테고리 | 기능 | 목적 |
|---------|------|------|
| **생성** | 프리셋 캐스팅, 신규 멤버 생성 | 만드는 재미 |
| **소비** | "다른 그룹" 콘텐츠 홈 노출 | 수요 측정 |
| **관계** | 1:1 채팅 | 감정적 유대 |
| **공유** | 프로필 URL 공유 | Phase 2 준비 |

**"다른 그룹" 콘텐츠 역할:**
- Alora가 만든 프리셋 AI 아이돌 콘텐츠
- 홈 화면에 노출하여 소비 욕구 측정
- "나도 만들고 싶다" 전환 유도

#### 3.1.4 성공 지표

**생성 지표:**
| 지표 | 목표 | 의미 |
|------|------|------|
| 그룹 생성 완료율 | 60%+ | 캐스팅 → 완료 |
| 평균 캐스팅 소요 시간 | 5분+ | 진지한 고민 |

**소비 지표 (가장 중요):**
| 지표 | 목표 | 의미 |
|------|------|------|
| "다른 그룹" 콘텐츠 클릭률 | 40%+ | 남의 AI 아이돌 관심 |
| 세션당 콘텐츠 탐색 개수 | 3개+ | 여러 콘텐츠 찾아봄 |
| 콘텐츠 완료율 | 70%+ | 끝까지 봄 |

**전환 지표 (Chicken-and-Egg 증명):**
| 지표 | 목표 | 의미 |
|------|------|------|
| 콘텐츠 소비 → 그룹 생성 전환율 | 30%+ | 보고 → 만들고 싶어짐 |
| 생성 → 프리미엄 전환율 | 10%+ | 더 보고 싶어서 결제 |
| 48시간 재방문율 | 50%+ | 다시 찾아옴 |

**공유 의향 (Phase 2 준비):**
| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 공유 의향 설문 점수 | 4.0+/5.0 | UT 설문 |
| "내 콘텐츠 보여주고 싶다" | 60%+ | UT 설문 |
| 공유 버튼 클릭 (기능 없어도) | 20%+ | 실제 행동 |

#### 3.1.5 Phase 2 전환 조건 (Go/No-Go)

**GO 조건 (Phase 2로 진행):**
```
소비 지표: 세션당 3개+ 콘텐츠 탐색
  AND
전환율: 30%+ 소비→생성
  AND
공유 의향: 4.0+ 점수
```

**해석:**
- ✅ 소비 지표 → AI 아이돌 콘텐츠 수요 증명
- ✅ 전환율 → Chicken-and-egg 해결 가능
- ✅ 공유 의향 → 크리에이터 공급 의지
- → **UGC 플랫폼 성공 확신**

**NO-GO (Pivot 필요):**
- 소비 지표 < 2개 → 콘텐츠 수요 없음
- 전환율 < 20% → Chicken-and-egg 못 풂
- 공유 의향 < 3.0 → 크리에이터 될 의향 없음

**Pivot 옵션:**
- Option A: B2C SaaS로 (개인 컴패니언 서비스)
- Option B: B2B 도구로 (소속사 대상)

#### 3.1.6 개발 우선순위

**P0 (MVP, 4주):**
- 그룹 생성 (캐스팅, 신규 멤버)
- 1:1 채팅
- **"다른 그룹" 콘텐츠 홈 노출** ⭐️ 가장 중요
- 프로필 공유

**P1 (Month 2-3):**
- 프리미엄 구독
- 콘텐츠 탐색 깊이 측정
- UT 설문 시스템

**P2 (Month 4-6):**
- 콘텐츠 추천 알고리즘
- 공유 버튼 (기능 없어도 클릭 측정)
- Phase 2 전환 준비

### 3.2 Phase 2: UGC 콘텐츠 플랫폼 (6-12개월)

#### 3.2.1 목표

**"바이럴 콘텐츠 폭발" + "크리에이터 이코노미 구축"**

#### 3.2.2 전환 트리거

- Phase 1 핵심 지표 달성
- 콘텐츠 자동 생성 시스템 완성
- 플랫폼 인프라 준비 (추천, 검색, 큐레이션)

#### 3.2.3 비즈니스 모델

```
유저 A가 AI 아이돌 생성
  ↓
콘텐츠 자동 생성 (AI 영상)
  ↓
플랫폼에 업로드 (공개)
  ↓
유저 B, C, D가 소비
  ↓
조회수/구독 기반 수익 분배
```

**수익 구조:**
- B2C Premium: $4.99/월 (광고 없음, 독점 콘텐츠)
- Creator Revenue Share: 70/30 split (크리에이터 70%)
- Marketplace: AI 아이돌 굿즈 판매 수수료 (15%)

#### 3.2.4 핵심 기능

| 카테고리 | 기능 | 유사 서비스 |
|---------|------|------------|
| **탐색** | 추천, 트렌딩, 검색 | YouTube 홈 |
| **소통** | 구독, 댓글, 좋아요, 공유 | 틱톡 |

| **수익화** | 광고, 멤버십, 굿즈 | YouTube 수익화 |

#### 3.2.5 성공 지표

| 지표 | 6개월 목표 | 12개월 목표 |

| 크리에이터 수익 (상위 100명 평균) | $100/월 | $500/월 |

#### 3.2.6 기술 과제

| 과제 | 솔루션 |

## 4. User Stories (전체 Phase)

### 4.1 Phase 1 Core User Stories

**US-P1-1: AI 아이돌 생성 (기존 유지)**
```
As a K-pop 팬,
I want to 나만의 AI 아이돌 그룹을 만들면,
So I can 내가 상상한 캐릭터를 소유할 수 있다.
```

**US-P1-2: 다른 그룹 콘텐츠 탐색 ⭐️ 신규**
```
As a 플랫폼 방문자,
I want to 다른 사람들이 만든(프리셋) AI 아이돌 콘텐츠를 보면,
So I can "나도 만들고 싶다"는 욕구를 느낄 수 있다.
```

**US-P1-3: 1:1 채팅 (기존 유지)**
```
As a 아이돌 생성자,
I want to 내 AI 아이돌과 대화하면,
So I can 감정적 유대를 형성할 수 있다.
```

**US-P1-4: 프로필 공유 (기존 유지)**
```
As a 아이돌 소유자,
I want to 내 AI 아이돌 프로필을 공유하면,
So I can 다른 사람들에게 자랑할 수 있다.
```

### 4.2 Phase 2 Core User Stories

**US-P2-1: 콘텐츠 업로드**
```
As a 크리에이터,
I want to 내 AI 아이돌의 자동 생성된 콘텐츠를 업로드하면,
So I can 다른 사람들에게 보여주고 수익을 얻을 수 있다.
```

**US-P2-2: 콘텐츠 탐색**
```
As a 소비자,
I want to 추천/트렌딩 AI 아이돌 콘텐츠를 탐색하면,
So I can 재미있는 콘텐츠를 발견할 수 있다.
```

**US-P2-3: 크리에이터 구독**
```
As a 팬,
I want to 좋아하는 크리에이터를 구독하면,
So I can 새 콘텐츠 알림을 받을 수 있다.
```

**US-P2-4: 수익 확인**
```
As a 크리에이터,
I want to 내 콘텐츠의 조회수와 수익을 확인하면,
So I can 어떤 콘텐츠가 인기인지 알 수 있다.
```

### 8.1 Product Risks

## 10. Success Scenarios

### 10.1 Best Case (Phase 1 → Phase 2 성공)

**6개월 후:**
```
전환율 40% → Chicken-and-egg 완전히 풀림
공유 의향 4.5+ → 크리에이터 대기 중
MAU 10K → Phase 2 출시 준비 완료
```

**12개월 후:**
```
MAU 100K
크리에이터 5K
일평균 업로드 300
바이럴 콘텐츠 발생 (1M+ 조회수)
```

### 10.2 Base Case (Phase 1 성공, Phase 2 느림)

**6개월 후:**
```
전환율 30% → Phase 2 전환 가능
공유 의향 4.0 → 크리에이터 의향 있음
MAU 5K → 천천히 성장
```

**12개월 후:**
```
MAU 30K
크리에이터 1K
UGC 플랫폼 작동하지만 느림
B2C 구독 수익으로 유지
```

### 10.3 Worst Case (Phase 1 실패)

**6개월 후:**
```
전환율 < 20% → Chicken-and-egg 못 풂
공유 의향 < 3.0 → 크리에이터 의향 없음
MAU < 1K → 제품 실패
```

**Pivot 옵션:**
- Option A: B2C SaaS (개인 컴패니언)
- Option B: B2B 도구 (소속사 대상)
- Option C: 종료
