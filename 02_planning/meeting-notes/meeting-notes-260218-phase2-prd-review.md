## 1. 요약

### 1.1 카카오벤처스 오피스아워 지원
- 카카오벤처스 오피스아워 지원 완료 — 제이가 GitHub 정리 자료 기반으로 슬라이드 제작
- 2/25~26 오피스아워 예정, 6팀 정도 참여하는 커피챗 형식

### 1.2 Phase 1 사용자 테스트 결과
- "내 새끼 같다", "잘생겼다" 등 의도한 반응 확인
- Sean Ellis 테스트(서비스 사라지면 매우 실망 40% 이상) 및 NPS(친구에게 추천/공유 30% 이상) 기준으로 PMF 검증 예정
- 사용자 인터뷰 인용을 IR 자료에 직접 포함하여 투자자 설득력 강화 계획

### 1.3 Phase 2 핵심 방향
- 목표: 채팅을 넘어 IP 육성에 개입하려는 행동이 있는지, 육성형 모델이 콘텐츠형 대비 더 높은 지불 의향을 보이는지 검증
- 로그인/회원가입 인증 필수 도입, 모바일 앱도 Phase 1 스프린트를 따라감

### 1.4 하이라이트 콘텐츠 구조
- is_premium 구분: 공개 콘텐츠(홈 화면 노출) vs 프리미엄 콘텐츠(구독자만 접근)
- 타 그룹 하이라이트 콘텐츠는 Phase 2에서도 홈 화면에 유지
- 콘텐츠 형식이 인터뷰 형식으로 통일된 상태 — 멤버 간 대화(염탐 느낌)와 질감이 다르므로 Phase 2에서 재논의 필요

### 1.5 채팅 내 육성 경험 설계 (수지 와이어프레임 초안)
- 채팅 안에서 "대화하기" vs "컨셉 정하기" 분기 선택지 다이얼로그 제공
- 컨셉 선택 시 프로듀서 대시보드 랜딩 (기획 완료/녹음 중/발매 예정 등 타임라인)
- 채팅 안에서 육성이 이루어지도록 설계한 이유: 서비스 방향이 확정 전이라 콘텐츠형으로도 전환 가능하게

### 1.6 개발 전 사전 검증 (페이월 프리뷰)
- 실제 육성 기능 개발 전에 페이월(랜딩 페이지)로 프리뷰만 보여주고 전환율/관심도 사전 검증
- 스프린트 4 목표로 설정

### 1.7 경쟁사 재정의
- 버블이 아닌 월간아이돌 등 육성형/캐릭터 채팅 서비스로 경쟁사 변경 필요
- 채팅으로 할 수 있는 부분과 게임적 부분의 영역 구분 필요

### 1.8 Phase 2 추가 과제
- 브랜딩 및 세계관 구축
- SNS 공식 계정 생성 및 운영
- 이미지 프롬프트 수정 (기존 아이돌 유사성 이슈 방지)
- 가격 플래닝 (경쟁사 분석 기반)
- 개인정보처리방침 / 이용약관
- Phase 1 수정사항 (UT 인터뷰 기반 피드백)

### 1.9 유료 전환 테스트
- 단일 가격이 아닌 복수 가격대(3천/5천/7천원) 앵커링 방식 적용
- UT 시 영업 방식으로 가격대별 반응 테스트

## 2. 결정 사항

1. Phase 2 핵심 방향: IP 육성형 서비스에 대한 사용자 반응(PMF) 검증
2. 채팅 내 분기 선택지(대화하기 vs 컨셉 정하기) 방식으로 육성 경험 제공
3. 실제 개발 전 페이월(프리뷰 랜딩 페이지)로 사전 검증 채택 (스프린트 4)
4. 경쟁사 분석 재정의: 버블 대신 월간아이돌 등 육성형 서비스로
5. 타 그룹 하이라이트 콘텐츠 홈 화면 노출 유지 (Phase 2에서도 계속)
6. 유료 전환 테스트 시 복수 가격대(3천/5천/7천원) 앵커링 방식 적용
7. 중간 회의(9시)에는 방향성만 러프하게 공유, 와이어프레임은 로우파이로
8. 내일(2/19) 저녁 9시: Phase 2 와이어프레임 논의 + IR 미팅
