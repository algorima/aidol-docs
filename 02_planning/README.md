# 02. Planning (기획)

PRD, 기능 명세서, 사용자 플로우 문서를 관리합니다.

## 폴더 구조

- `prd/` - Product Requirements Document (제품 요구사항 문서)
- `feature-specs/` - 기능별 상세 명세서
- `user-flows/` - 사용자 플로우 및 시나리오

## 파일명 규칙

```
YYYYMMDD__주제__상태__vX.Y__소유자.md
```

예시:
- `20260115__aidol-prd__draft__v0.1__yw.md`
- `20260120__chat-feature-spec__review__v0.3__je.md`
