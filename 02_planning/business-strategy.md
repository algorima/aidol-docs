# AIoIA 비즈니스 전략

## 1. 핵심 비전

### AI 시대의 유튜브/틱톡

> "AI 아이돌이 넘쳐나고, 콘텐츠가 폭발하고, 바이럴이 일어나는 플랫폼"

**우리는:**
- ❌ K-pop 특화 Character.AI가 아님
- ❌ 1:1 AI 컴패니언 서비스가 아님
- ✅ **AI 아이돌 콘텐츠 UGC 플랫폼**

**최종 목표:**
```
수많은 유저가 AI 아이돌 생성
  ↓
AI 아이돌 콘텐츠 대량 생산
  ↓
다른 유저들이 소비
  ↓
바이럴 콘텐츠 발생
  ↓
플랫폼 성장 (유튜브/틱톡 모델)
```

### 1.1 시장 규모 (TAM/SAM/SOM)

**TAM (Total Addressable Market): ~$30B**
- 글로벌 UGC 콘텐츠 플랫폼 시장
- 참고: YouTube ($31B, 2023), TikTok ($14B, 2023), Character.AI (추정 $1B+ valuation)
- AI 생성 콘텐츠는 기존 UGC 시장을 확장 (낮은 진입장벽 → 더 많은 크리에이터)

**SAM (Serviceable Available Market): ~$3B**
- AI 캐릭터 인터랙션 시장
  - Character.AI: 20M MAU, 2시간+ 평균 세션 (2024)
  - Replika: 10M+ downloads
  - 버추얼 인플루언서 시장: $5B (2025 예상)
- K-pop 팬덤 시장 (글로벌)
  - K-pop 글로벌 팬덤: 178M (2023, Circle Chart)
  - 평균 팬당 연간 지출: $150-300 (앨범, 굿즈, 콘서트)

**SOM (Serviceable Obtainable Market): ~$100M (Year 3 목표)**
- Phase 2 성공 시 목표 (3년 내)
  - MAU 200K (Year 1) → 1M (Year 2) → 5M (Year 3)
  - 크리에이터 10K → 50K → 200K
  - ARPU $5/월 (구독 + 수익 분배)
  - 5M MAU × 20% 결제율 × $5/월 × 12개월 = $60M ARR
  - 크리에이터 수익 분배 30% 추가 = ~$100M

**시장 기회:**
- Character.AI는 1:1 채팅만, 바이럴 콘텐츠 없음 → AIoIA는 UGC 플랫폼
- YouTube/TikTok은 진입장벽 높음 → AIoIA는 AI 자동 생성
- K-pop 시장은 소수의 그룹에 집중 → AIoIA는 무한한 AI 아이돌

## 2. 단계별 전략

### 2.1 Phase 1: 개인 AI 컴패니언 모델 (3주, Sprint 1-3)

**Why Phase 1이 필요한가?**
- 콘텐츠 대량 생산 기술이 아직 부족 (자동 생성 or 노가다 둘 다 어려움)
- Chicken-and-egg 문제 해결 필요 (초기 콘텐츠 공급)
- 핵심 가설 검증: "유저가 AI 아이돌 만드는 게 재미있는가?"

**비즈니스 모델:**
```
유저가 AI 아이돌 그룹 생성 (캐스팅)
  ↓
프리셋 or 커스텀 AI 아이돌
  ↓
1:1 채팅 + 채팅 UI 콘텐츠 소비 (멤버 후기, 숙소 썰)
  ↓
프리미엄 콘텐츠 구독 ($3.99/월)
```

**초기 콘텐츠 공급:**
- Alora가 프리셋 AI 아이돌 그룹 제공
- "다른 그룹" 콘텐츠로 홈 화면에 노출
- 유저가 참고하며 자기 그룹 생성

**검증할 핵심 가설:**
1. **생성의 재미:** 유저가 AI 아이돌 만드는 걸 즐기는가?
2. **소비 욕구:** 다른 AI 아이돌 콘텐츠를 보고 싶어하는가?
3. **전환 가능성:** 콘텐츠 소비 → 생성으로 전환되는가?
4. **공유 의향:** 내 AI 아이돌 콘텐츠를 공유하고 싶어하는가?

### 2.2 Phase 2: UGC 콘텐츠 플랫폼 (6-12개월)

**전환 조건:**
- Phase 1 핵심 지표 달성 (섹션 3 참고)
- 콘텐츠 자동 생성 시스템 완성
- 플랫폼 인프라 준비 (추천, 검색, 큐레이션)

**비즈니스 모델:**
```
유저 A가 AI 아이돌 생성
  ↓
AI 아이돌 콘텐츠 자동 생성
  ↓
플랫폼에 업로드 (공개)
  ↓
유저 B, C, D가 소비
  ↓
조회수/구독 기반 유저 A 수익 분배
```

**수익 모델:**
- B2C: 프리미엄 구독 (광고 없음, 독점 콘텐츠)
- Creator Economy: 크리에이터 수익 분배 (70/30 split)
- Marketplace: AI 아이돌 굿즈, MD 판매 수수료

**플랫폼 기능:**
- 콘텐츠 탐색 (추천, 트렌딩, 검색)
- 구독/팔로우 시스템
- 댓글/좋아요/공유
- 크리에이터 대시보드 (분석, 수익 관리)

### 2.3 Phase 3: B2B 확장 (12개월+)

**확장 방향:**

**2.3.1. IP 라이센싱 (우선순위 1위)**
- **타겟:** 소상공인, 1인 브랜드 운영자
  - 아이디어스(idus) 개인 사업자
  - 스마트스토어 셀러
  - 소규모 온라인 쇼핑몰
- **가치 제안:** 모델 섭외 비용 없이 AI 아이돌을 제품 모델로 활용
- **비즈니스 모델:**
  - 플랫폼에서 인기를 얻은 AI 아이돌 IP를 브랜드에 라이센싱
  - 월 구독 or 건별 과금 (예: 월 $29 or 이미지당 $5)
  - 크리에이터-플랫폼-브랜드 수익 분배 (예: 50/30/20)
- **사용 사례:**
  - 의류 쇼핑몰: AI 아이돌이 옷 착용 모델
  - 액세서리 브랜드: AI 아이돌이 제품 홍보
  - 화장품: AI 아이돌 뷰티 콘텐츠

**2.3.2. 소속사/MCN 파트너십**
- 공식 AI 아이돌 출시
- 플랫폼 화이트라벨

**2.3.3. 크리에이터 도구 라이센스**
- AI 아이돌 생성 API
- 콘텐츠 자동화 도구

**Note:** Phase 3는 확장 전략이지 핵심 비즈니스가 아님

## 3. Phase 1 핵심 검증 지표

### 3.1 생성 지표 (Creation Metrics)

| 지표 | 목표 | 의미 |
|------|------|------|
| 그룹 생성 완료율 | 60%+ | 캐스팅 시작 → 완료 |
| 평균 캐스팅 소요 시간 | 5분+ | 진지하게 고민하는가 |
| 멤버 선택 변경 횟수 | 2회+ | 신중하게 선택하는가 |

### 3.2 소비 지표 (Consumption Metrics) ⭐️ 가장 중요

| 지표 | 목표 | 의미 |
|------|------|------|
| "다른 그룹" 콘텐츠 클릭률 | 40%+ | 남의 AI 아이돌에 관심 있는가 |
| 콘텐츠 평균 체류 시간 | 3분+ | 끝까지 보는가 |
| 세션당 콘텐츠 탐색 개수 | 3개+ | 여러 콘텐츠를 찾아보는가 |
| 콘텐츠 완료율 | 70%+ | 중간에 이탈하지 않는가 |

### 3.3 전환 지표 (Conversion Metrics) ⭐️ Chicken-and-Egg 증명

| 지표 | 목표 | 의미 |
|------|------|------|
| 콘텐츠 소비 → 그룹 생성 전환율 | 30%+ | 보고 → 만들고 싶어지는가 |
| 생성 → 프리미엄 전환율 | 10%+ | 더 보고 싶어서 결제하는가 |
| 콘텐츠 → 재방문율 (48시간) | 50%+ | 다시 찾아오는가 |

### 3.4 공유 의향 (Sharing Intent) ⭐️ Phase 2 준비

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 공유 의향 설문 점수 | 4.0+/5.0 | UT 설문 |
| "내 콘텐츠 보여주고 싶다" | 60%+ | UT 설문 |
| 공유 버튼 클릭 (기능 없어도) | 20%+ | 실제 행동 측정 |

### 3.5 성공 기준

**Phase 2로 전환 가능 조건:**
```
소비 지표 높음 (3개+ 콘텐츠 탐색)
  AND
전환율 높음 (30%+ 소비→생성)
  AND
공유 의향 높음 (4.0+ 점수)
```

**해석:**
- ✅ 소비 지표 높음 → 수요 증명 (AI 아이돌 콘텐츠에 대한 갈증)
- ✅ 전환율 높음 → Chicken-and-egg 해결 가능
- ✅ 공유 의향 높음 → 공급 의지 증명
- → **Phase 2 (UGC 플랫폼) 성공 확신** 🚀

## 4. Chicken-and-Egg 해결 전략

### 4.1 문제 정의

**전형적인 양면 플랫폼 Chicken-and-Egg:**
- 콘텐츠 없으면 → 소비자 안 옴
- 소비자 없으면 → 크리에이터 안 만듦

### 4.2 우리의 해법

**Phase 1: Seed + Validate**
```
1. Alora가 프리셋 AI 아이돌 제공 (초기 공급)
   ↓
2. 유저가 프리셋 콘텐츠 소비
   ↓
3. "나도 만들고 싶다" (생성)
   ↓
4. 공유 의향 측정
```

**Phase 2: Loop**
```
유저 A가 AI 아이돌 생성
  ↓
콘텐츠 자동 생성 + 업로드
  ↓
유저 B가 소비
  ↓
"나도 만들고 싶다"
  ↓
유저 B가 크리에이터로 전환
  ↓
더 많은 콘텐츠 생성
  ↓
플랫폼 성장 (network effect)
```

### 4.3 핵심 차별점

**vs 다른 UGC 플랫폼 (유튜브/틱톡):**
- 유튜브: 영상 제작 진입장벽 높음
- AIoIA: AI가 콘텐츠 자동 생성 → **진입장벽 낮음**

**vs AI 챗봇 (Character.AI):**
- Character.AI: 1:1 채팅만 (콘텐츠 소비 없음)
- AIoIA: 채팅 UI 콘텐츠 + 공유 → **바이럴 가능**

## 5. 현재 Sprint 2-3 재해석

### 5.1 Sprint 2 목표 (재정의)

**기존 KR:**
```
콘텐츠 상세 → 결제 전환율 10% 이상
```

**진짜 의미:**
```
"다른 그룹" 콘텐츠 소비
  ↓
"나도 만들고 싶다" (그룹 생성)
  ↓
"내 그룹 콘텐츠 더 보고 싶다" (프리미엄 결제)
```

**추가 KR (필수):**
- **콘텐츠 소비 → 그룹 생성 전환율 30% 이상**
- **세션당 콘텐츠 탐색 개수 3개 이상**
- **콘텐츠 완료율 70% 이상**

### 5.2 Sprint 3 UT 설문 추가

**기존 설문 외 추가:**

**1. 소비 욕구 측정**
> Q. 다른 사람들이 만든 AI 아이돌 콘텐츠를 더 보고 싶다 (1-5)

**2. 공유 의향 측정**
> Q. 내 AI 아이돌 그룹의 콘텐츠를 다른 사람들과 공유하고 싶다 (1-5)

**3. 수익 관심 측정**
> Q. 만약 내가 만든 AI 아이돌 콘텐츠가 인기를 얻어 수익이 생긴다면?
> - 매우 흥미 있음
> - 흥미 있음
> - 보통
> - 관심 없음

**4. Phase 2 전환 가능성**
> Q. AI 아이돌 콘텐츠 플랫폼 (유튜브 같은)이 있다면 사용하겠는가?
> - 크리에이터로 참여 (AI 아이돌 만들어 업로드)
> - 소비자로 참여 (다른 사람 콘텐츠 구경)
> - 둘 다
> - 관심 없음

### 6.2 핵심 메시지

**투자자에게 말하는 스토리:**

> "우리는 AI 시대의 유튜브를 만들고 있습니다.
>
> 하지만 유튜브와 달리, AI가 콘텐츠를 자동으로 만들어주기 때문에
> 크리에이터 진입장벽이 낮습니다.
>
> Phase 1에서 '만드는 재미'와 '소비 욕구'를 검증하고,
> Phase 2에서 본격적인 UGC 플랫폼으로 전환합니다.
>
> Character.AI는 1:1 채팅의 한계에 부딪혔지만,
> 우리는 채팅 UI 콘텐츠와 공유를 통해 바이럴을 만들어냅니다."

### 6.3 리스크와 대응

**리스크 1: Phase 1에서 실패하면?**
- Phase 1은 최소 비용으로 검증 (프리셋만 제공)
- 실패해도 피벗 가능 (B2B 도구로 전환)

**리스크 2: 콘텐츠 자동 생성 기술이 안 되면?**
- Phase 1 검증 동안 기술 개발 병행
- 최악의 경우 크리에이터 직접 제작 (큐레이션 플랫폼)

**리스크 3: 경쟁사가 먼저 하면?**
- K-pop 팬덤 20년 경험 = 핵심 차별점
- UX/HCI 전문성으로 진입장벽 확보
