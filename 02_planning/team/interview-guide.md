# 신입 팀원 인터뷰 질문 가이드

**작성일**: 2026-03-04

---

## 기술 및 경험

```
□ 현재 주력 기술 스택은 무엇인가요?

□ 팀 프로젝트 경험이 있나요?
  └─ 어떤 역할을 맡았나요?

□ Git 브랜치/머지/PR 경험이 있나요?
  └─ 어떻게 사용하셨나요?

□ 포트폴리오나 블로그를 운영하시나요?
```

---

## AI/LLM 관련 (해당 시)

```
□ AI/LLM을 어떻게 활용하고 있나요?
  └─ 어떤 도구를 사용하시나요?

□ AI의 한계를 어떻게 인식하고 있나요?

□ 코드 작성 시 AI를 어떻게 활용하나요?
  └─ AI가 제안한 코드를 어떻게 검증하나요?
```

---

## 시간 및 참여

```
□ 현재 학년/직장 상황은 어떻게 되시나요?

□ 평일과 주말에 투입할 수 있는 시간은?

□ 오프라인 참여가 가능한가요?
  └─ 어느 지역에 거주하시나요?

□ 프로젝트 기간은 어느 정도 생각하시나요?
```

---

## 복수 프로젝트 및 시간 관리

```
□ 현재 진행 중인 다른 프로젝트나 활동이 있나요?
  └─ 어떤 프로젝트인가요? (규모, 역할, 기간)
  └─ 언제까지 진행되나요?
  └─ 이 프로젝트와의 우선순위는 어떻게 되나요?
  └─ 주당 투입 시간 분배 계획은?

□ 그 시간 동안 실제로 무엇까지 가능한가요?
  └─ PR 작성? 코드 리뷰? 둘 다?

□ 리뷰 요청을 받으면 언제까지 응답할 수 있나요?
  └─ 평일? 주말? 어느 쪽을 선호하시나요?

□ 작업 시작 전 팀원과 공유하는 습관이 있나요?
  └─ 어떤 도구를 사용하시나요? (Asana, Slack 등)
```

```

□ 이 프로젝트에서 무엇을 얻고 싶으신가요?

□ 가볍게 vs 몰입형 중 어느 쪽을 선호하시나요?
```
