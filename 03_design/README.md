# 03. Design (디자인)

브랜딩, UI 컴포넌트, 프로토타입 문서를 관리합니다.

## 폴더 구조

- `branding/` - 브랜드 가이드라인, 로고, 컬러 팔레트
- `ui-components/` - UI 컴포넌트 명세 및 스타일 가이드
- `prototypes/` - Figma 프로토타입 링크 및 설명

## 파일명 규칙

```
YYYYMMDD__주제__상태__vX.Y__소유자.md
```

예시:
- `20260201__brand-guidelines__final__v1.0__sy.md`
- `20260205__color-palette__review__v0.2__je.md`
