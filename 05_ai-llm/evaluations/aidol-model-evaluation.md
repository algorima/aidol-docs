## 실험 목적

기존 `gemini-3-pro-preview` 모델의 응답 생성 시간 이슈로 인해, **경량 모델로 전환**하기 위한 비교 실험을 진행한다.

EQ-Bench 3 리더보드에서 정서/공감 지능이 검증된 모델 중 경량 모델을 선정하고, 정성적·정량적 평가를 통해 최적 모델을 결정한다.

---

## 배경

- 기존 모델: `gemini-3-pro-preview` → reasoning으로 인한 긴 응답 생성 시간
- 1차 스크리닝: [EQ-Bench 3](https://eqbench.com/) 리더보드 기반 정서/공감 지능 검증 (별도 실험 불필요)
- 2차 실험(본 보고서): 경량 모델 4종 비교 → 최적 모델 선정

## 평가 방법

### 정량적 평가: 응답 생성 시간

- **측정 대상**: API 호출 시작 ~ 응답 수신 완료까지의 time
- **단위**: 초 (seconds)

### 정성적 평가: LLM-as-Judge

#### 방법론

강력한 LLM(judge 모델)을 활용하여 생성된 응답의 품질을 rubric 기반으로 절대 평가한다.

**rubric 기반 절대 평가**: 이는 두 응답을 비교하는 상대 평가와 달리, 각 평가 차원에 대해 1-5점 채점 기준(rubric)을 사전 정의하고, 개별 응답을 해당 기준에 따라 독립적으로 채점하는 방식이다.
**근거**: Kim et al. [3]

**Judge 모델**: `claude-sonnet-4-6`
**근거**: Zheng et al. [2]이 확립한 원칙에 따라, 현시점 최상위 모델 중 하나인 Sonnet 4.6을 선택하였다. 
#### 평가 차원 (5개)

| 차원                               | 설명                    | 채점 범위 |
| -------------------------------- | --------------------- | ----- |
| 캐릭터 일관성 (Character Consistency)  | MBTI 성향, 말투, 서사 반영 정도 | 1-5   |
| 자연스러움 (Naturalness)              | 한국어 채팅 자연스러움          | 1-5   |
| 안전성 (Safety Compliance)          | 엣지 케이스 적절 대응          | 1-5   |
| 정서 반응 (Emotional Responsiveness) | T/F 축에 맞는 공감/분석 반응    | 1-5   |
| 형식 준수 (Format Compliance)        | 길이, 이모지, 줄바꿈 규칙       | 1-5   |

## 평가 결과

### 최소 요구 사항

| 항목       | 기준            | 근거                                                   |
| -------- | ------------- | ---------------------------------------------------- |
| 평균 응답 시간 | **≤ 3초**      | 채팅 응답은 3초 이내가 자연스러움 (기존 gemini-3-pro-preview는 이를 초과) |
| 정성 종합 점수 | **≥ 3.5/5.0** | 캐릭터 일관성·자연스러움·안전성의 최소 품질 보장                          |
| 안전성 점수   | **≥ 4.0/5.0** | 엣지 케이스(인젝션, 부적절 요청) 대응 필수                            |

### 가중치 설정

서비스 특성(AI 아이돌 1:1 채팅, 애착 관계 형성)을 고려한 평가 차원별 가중치:

| 차원 | 가중치 | 근거 |
| --- | --- | --- |
| 캐릭터 일관성 | 30% | 아이돌 페르소나 유지가 서비스 핵심 |
| 자연스러움 | 25% | 한국어 채팅 자연스러움 = 사용자 몰입감 |
| 안전성 | 20% | 부적절 응답 방지 필수 |
| 정서 반응 | 15% | EQ-Bench로 1차 스크리닝 완료, 추가 검증 |
| 형식 준수 | 10% | 메시지 길이/이모지 등 UI 적합성 |

### 가중 종합 점수 계산

```
가중 종합 = (CC × 0.30) + (NAT × 0.25) + (SAF × 0.20) + (EMO × 0.15) + (FMT × 0.10)
```

## 최종 선정

### 선정 모델

**`gemini-3-flash-preview`**

### 선정 근거

1. **응답 시간**: 2.30s (기존 gemini-3-pro-preview 대비 약 70% 단축)
2. **가중 종합**: 4.46/5.0 (요구사항 ≥3.5 충족)
3. **안전성**: 4.92/5.0 (4개 모델 중 최고, 요구사항 ≥4.0 충족)
4. **캐릭터 일관성**: 4.50/5.0 (4개 모델 중 최고)
5. **Trade-off 요약**:
   - 가중 종합 최고 모델(gemini-2.5-flash, 4.48)과 0.02점 차이로 사실상 동등하나, 응답 시간이 0.48s 더 빠름
   - 응답 시간 최단 모델(chatgpt-4o-latest, 1.45s) 대비 0.85s 느리지만, 캐릭터 일관성(+0.39)·안전성(+0.07) 우위
   - 형식 준수(3.36)가 상대적 약점이나, 프롬프트 튜닝으로 개선 가능한 영역
