## 일별 세부 계획

### 3/8 (일) — Off / 준비

- 팀 PRD 리뷰 (수지+소연 진행)
- 채현은 공식 off이지만, 월요일 실험에 쓸 레퍼런스 이미지 수집 가능 (선택)
  - 참고할 케이팝 컨셉 포토 5~10장 북마크
  - Gemini API 키·환경 세팅 최종 확인

### 3/9 (월) — 블록 구조 초안 + 핵심 조합 실험

**오전 (설계)**
- 프롬프트 블록 구조 초안 작성
  - 블록 후보: `[인종/피부톤]` `[분위기/컨셉]` `[인상/이목구비]` `[헤어스타일/컬러]` `[의상]` `[배경]` `[카메라/조명]`
  - 각 블록별 파라미터 후보값 3~5개씩 리스트업
  - 블록 간 조합 규칙 정의 (어떤 순서로 이어붙일지)

**오후~저녁 (실험)**
- 핵심 조합 5~6개 선정 → 이미지 생성
  - 모델: `gemini-3.1-flash-image-preview`
  - 해상도: 1K (빠른 탐색용)
  - 목표 생성량: ~20장
  - 조합 예시:
    1. 남성 / 시크 / 날카로운 인상 / 블랙 숏컷
    2. 남성 / 청량 / 부드러운 인상 / 브라운 미디엄
    3. 여성 / 걸크러시 / 강한 눈매 / 블론드 롱
    4. 여성 / 청순 / 둥근 이목구비 / 블랙 롱
    5. 남성 / 다크 / 묵직한 인상 / 실버 숏컷
    6. 여성 / 큐트 / 동글동글 / 핑크 밥컷
- 생성 결과에 파일명 규칙 적용: `w1_{조합번호}_{시도번호}.png`

**기록**
- 각 조합별 사용한 프롬프트 전문 + 결과 이미지 매칭해서 로컬 정리

### 3/10 (화) 낮 — 상위 조합 확장 + 4K 확인

- 월요일 결과 중 상위 조합 3~4개 선별
  - 선별 기준: 아이돌 비주얼 적합성, 캐릭터 일관성 가능성
- 상위 조합을 4K로 업스케일 생성 (~10장)
  - 4K에서 디테일 깨지는 부분 확인 (피부 질감, 머리카락, 눈)
  - 비용 추적: 장당 ~$0.15 × 10장 = ~$1.5

### 3/10 (화) 밤 — 슬랙 공유

`#6_ai_llm_프롬프트_채현` 채널에 실험 결과 공유

```
📌 Week 1 — 캐스팅 이미지 블록 구조 첫 실험

가설: 인종/분위기/인상/헤어 4블록 조합으로 아이돌 비주얼 구현 가능한지 확인
조건: gemini-3.1-flash-image-preview, 1K→4K, 6개 조합
결과 이미지: [상위 3~5장 첨부]
소감: [뭐가 됐고 뭐가 안 됐는지]
다음: [수요일 수정 방향]

@미진 @제이 피드백 부탁드립니다!
```

### 3/10~3/11 — 미진·제이 비동기 피드백

- 미진: 팬덤·서사 관점 — "아이돌처럼 보이나?", 덕질 감성에 맞는지
- 제이: 비주얼·컨셉 관점 — 스타일링, 배경 처리, 전반적 퀄리티
- 피드백 형식: 슬랙 스레드에 이미지별 코멘트

### 3/11 (수) 낮 — 피드백 반영 프롬프트 수정

- 미진·제이 피드백을 항목별로 정리
- 프롬프트 수정 포인트 도출
  - 예: "피부톤이 너무 밝다" → 피부톤 블록 파라미터 조정
  - 예: "서양 잡지 느낌" → 조명·배경 블록 수정
- 수정된 프롬프트로 재생성 (5~10장)
