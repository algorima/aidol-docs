# Buppy/AIoIA 사용자 테스트 버그 리포트

2026년 2월 22일 IR 발표 피드백 미팅에서 진행된 사용자 테스트 중 발견된 에러/버그 및 UX 이슈를 정리한 문서입니다.

## 1. Error / Bug

### 1.1 이미지 생성 반복 에러 (프롬프트 길이 제한)

- 프롬프트가 조금만 길어지면 이미지 생성이 실패
- 10회 이상 반복 발생하여 데모 시연 불가 수준
- 백엔드에서 에러 발생하나, 사용자에게 원인 안내 없음
- "이거는 꼭 리포팅 할게. 큰일 날 뻔... 이미지 생성 하나 못하는 서비스라며"

### 1.2 성별 선택이 이미지 생성에 반영되지 않음

- 남성을 선택했지만 "청순한" 등 키워드에 따라 여성 이미지가 생성됨
- 반복적으로 발생하는 문제
- "내가 남성을 선택했는데 저렇게 청소한 이런 거 하면은 여자가 나와"

### 1.3 에러 메시지 / 원인 안내 부재

- 이미지 생성 실패 시 사용자에게 원인이나 안내가 전혀 없음
- 글자 수 제한에 대한 고지도 없음
- "왜 생성이 안 되는지라도 알려주시던가요? 심각하다"
- "몇 자 이상 이하로 입력해주세요 이런 거라도 넣어야지"

### 1.4 한글 프롬프트 입력 시 에러

- 한글로 프롬프트 입력 시 에러 발생
- 한글을 영어로 자동 변환하는 기능 없음
- "이렇게 적으면 이렇게 전환해 줄 수 있는 걸 백핸드에서 넣든가 해야 될 것 같은데. 그거 안 되면 이거 서비스 못해"

### 1.5 계약 상태 잘못 표시

- 계약 완료가 아니어야 하는 멤버가 계약 완료로 표시됨
- "왜 계약 완료야? 계약 완료되면 안 되는데 원래 저렇게"

### 1.6 버튼 미표시 버그

- 멤버 케미 관련 화면에서 있어야 할 버튼이 렌더링되지 않음
- "야 근데 저거 버튼이 있어야 되는데 왜 버튼이 없냐? 이거 진짜 환장하겠네"

### 1.7 잘못된 이미지 삭제 불가

- AI가 실제 아이돌과 비슷하게 생성한 이미지를 삭제하는 기능이 작동하지 않음
- "잘못된 것들도 다 삭제 지금 안 돼"

### 1.8 공유하지 않으면 사진 유실

- 생성한 아이돌 프로필을 공유하지 않으면 데이터가 사라짐
- "이거 공유 안 하면은 사진 날아가거든"

## 2. UX Issue

### 2.1 네비게이션 구조 부재 (바텀 바 없음)

- 하단 내비게이션 바가 없어서 홈 복귀 불가
- 캐스팅 보드 접근도 어려움, 뒤로 가기만 반복해야 함
- "홈에 돌아갈 수가 없는데 뒤로 뒤로 뒤로"
- "아니 이거를 바텀 바를 해놨어야지. 이거를 왜 계속 유지를 안 시켰대?"

### 2.2 영어 전용 프롬프트 (한글 미지원)

- 프롬프트를 영어로만 입력 가능하여 한국어 사용자에게 큰 진입 장벽
- 한글 입력을 영어로 자동 변환하는 기능 필요
- "잠깐만 영어가 어렵네요"
- "한글을 이렇게 변화할 수 있는 걸로 좀 해야 될 것 같아"

### 2.3 프롬프트 예시 / 템플릿 부재

- 사용자가 어떤 프롬프트를 입력해야 하는지 가이드/예시 없음
- 시스템 프롬프트에 기본 키워드(K-pop idol portrait 등)도 미설정
- "예시를 넣어야 되는데 못 넣어놨잖아"
- "케이팝 아이돌 이런 거는 미리 시스템 프롬프트로 넣어놔야지"

## 3. 우선순위 요약

### P0 (치명적)

1. 이미지 생성 반복 에러 (프롬프트 길이 제한)
2. 에러 메시지 부재
3. 한글 프롬프트 미지원
4. 네비게이션 부재 (바텀 바 없음)

### P1 (높음)

5. 성별 선택 무시 버그
6. 프롬프트 예시/템플릿 부재
