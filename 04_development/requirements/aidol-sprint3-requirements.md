# AIdol Sprint 3 - 프론트엔드 요구사항

## 1. 채팅방 목록 페이지

### 1.1 채팅 대상 멤버 (1명)

| 항목               | 필드명            | 설명                         |
| ------------------ | ----------------- | ---------------------------- |
| 멤버 이름          | `name`            | 멤버 표시 이름               |
| 멤버 ID            | `id`              | 멤버 고유 식별자             |
| 프로필 사진        | `profileImageUrl` | 멤버 프로필 이미지 URL       |
| 멤버 현재 상태     | -                 | 현재 시간에 따른 멤버의 상태 |
| 마지막 메시지 내용 | -                 | 가장 최근 수신/발신 메시지   |
| 마지막 메시지 시간 | -                 | 마지막 메시지의 수신 시각    |

- 멤버 현재 상태 값: `휴식 중`, `연습 중`, `잠깐 휴식 중`, `집에서 휴식 중`

### 1.2 나머지 멤버들

| 항목        | 필드명            | 설명                   |
| ----------- | ----------------- | ---------------------- |
| 멤버 ID     | `id`              | 멤버 고유 식별자       |
| 이름        | `name`            | 멤버 표시 이름         |
| 프로필 사진 | `profileImageUrl` | 멤버 프로필 이미지 URL |

## 2. 채팅방 페이지

### 2.1 채팅 대상 멤버 (1명)

| 항목           | 필드명            | 설명                         |
| -------------- | ----------------- | ---------------------------- |
| 멤버 이름      | `name`            | 멤버 표시 이름               |
| 프로필 사진    | `profileImageUrl` | 멤버 프로필 이미지 URL       |
| 멤버 현재 상태 | -                 | 현재 시간에 따른 멤버의 상태 |

- 멤버 현재 상태 값: `휴식 중`, `연습 중`, `잠깐 휴식 중`, `집에서 휴식 중`

### 2.2 채팅 메시지 리스트

| 항목         | 필드명        | 타입                      | 설명                    |
| ------------ | ------------- | ------------------------- | ----------------------- |
| 메시지 ID    | `id`          | `string`                  | 메시지 고유 식별자      |
| 보낸 사람    | `senderType`  | `"user"` \| `"companion"` | 메시지 발신자 유형      |
| 보낸 멤버 ID | `companionId` | `string` \| `null`        | 메시지를 보낸 멤버의 ID |
| 메시지 내용  | `content`     | `string`                  | 메시지 본문             |
| 보낸 시간    | `createdAt`   | `string (ISO 8601)`       | 메시지 전송 시각        |
